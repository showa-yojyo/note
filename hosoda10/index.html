Python 入門［２＆３対応］読書ノート¶

Python 関連の本を持っていないと思って、書店でテキトーに選んで買ったもの。入門書を謳っているのに話題が豊富。物量的にも 500 ページちょいとなって CP がよい。小手先テクニックでよければ、書いてあるコードを自分で再現するだけでもかなり身に付く。
