LibreOffice 関連資料¶

LibreOffice

本拠地ホームページ。プログラム本体だけでなく利用者手引も入手するべし。

libreofficehelp.com

ブログ構造のチュートリアル。公式ガイドを読む前に習得していくのもアリだ。

インストール方法¶

公式サイトからインストーラーをダウンロードして実行するのもよいが、本ノートではパッケージ管理ツール winget の使用を推奨する。

winget install -e --id TheDocumentFoundation.LibreOffice

See also

Windows Package Manager CLI 利用ノート

個人的必修技法¶

  • LibreOffice のバージョン (Help ‣ About LibreOffice)

  • オプションダイアログは Alt + F12 で開く

  • キーバインド確認方法 (Tools ‣ Customize…)

  • UI をチュートリアル推奨様式に変更

以下、各アプリケーション個別設定になる。

  • パスワード付与保存 (File ‣ Save As…)

  • 既定雛形変更 (File ‣ Templates ‣ Manage Templates…)

個人的設定オプション¶

ほとんどの項目で既定値が良い。自分好みの値に変更するものと、既定値のままであっても意識するべき構成を以下に記す。

  • LibreOffice

    • General

      • Help

        • Extended tips をオン

        • Warn if local help is not installed はオフ

        • Show “Tip of the Day” dialog on start-up をオン

      • Open/Save Dialogs

        • Use LibreOffice dialogs をオフ

      • Help Improve LibreOffice

        • Send crash reports to The Document Foundation をオン

    • Security

      • Security Options and Warnings の Options… ボタン

        • Remove personal information on saving をオン

    • Online Update

      • Online Update Options

        • Check for updates automatically は Every day

      • User Agent

        • Send OS version and basic hardware information をオン

  • Load/Save

    • Save

      • Save AutoRecovery information every をオンにして何分でもいいから指定

      • Always create backup copy をオフ

  • Language Settings

    • Languages

      • Language Of

        • User interface を English (USA) に

      • Formats

        • Date acceptance patterns から自分が使わないものを除外する

    • Asian Layout については既定値よりも良い設定がある可能性がある

設定バックアップ方法¶

Options ダイアログで LibreOffice ‣ Path をクリックすると Paths used by LibreOffice 一覧が示される。このうち、パスが

%APPDATA%\LibreOffice\4\user

のフォルダー以下が LibreOffice カスタマイズ設定置場と考えられる。したがって、ここはバックアップをするべきだろう。Git によるバージョン管理をするなり、オンラインストレージへの同期を定期的に実施するなり、対応しろ。

個人的必修キーバインド共通編¶
