6.3 常微分方程式の解の存在と一意性（基本）¶

  • ベクトル場 \(X\) に対して次のような \(F_t\) が存在するのかを考える： \(\fn{F_t}{M}M,\ {F_s \circ F_t} = F_{s + t}.\)

  • これは多様体上で常微分方程式を考える問題だ。

  • 開集合 \({U \subset \RR^n}\) とコンパクト集合 \({K \subset U}\) があると仮定すれば、\({\forall \bm x \in K, \exists B_\eps(\bm x) \subset U}\) が成り立つ。コンパクト性から \({\eps \gt 0}\) の取り方は \({\bm x \in K}\) に依存しないというのがポイントだ。

  • 定理 6.3.1 常微分方程式の解の存在、一意性、初期値に対する連続性

    • 仮定

      • リプシッツ条件： \(\exists L \gt 0 \text{ s.t. } t \in (a, b),\ \bm x_1, \bm x_2 \in U: \norm{X(t, \bm x_1) - X(t, \bm x_2)} \le L \norm{\bm x_1 - \bm x_2}.\)

      • 有界性： \(\fn{X}{(a, b) \times U}\RR^n\) は有界連続とする： \({\displaystyle \sup_{t, \bm x \in (a, b) \times U} X(t, \bm x) \le M.}\)

        • \(M\) の記号がカブっている。

    • 結論

      \(\exists \eps_0 \gt 0, \exists \fn{F}{(t_0 - \eps_0, t_0 + \eps_0) \times K}U,\)

      • \({F(t, \bm x)}\) は \(t\) について微分可能であり \(\bm x\) について連続。

      • \({\displaystyle F(t_0, \bm x) = \bm x,\ \diff{F}{t} = X(t, F(t, \bm x)).}\)

    • 証明

      • そのある関数とは \({\displaystyle F(t, \bm x) = \bm x + \int_{t_0}^t \diff{F(s, \bm x)}{s}\ \dd{s} = \bm x + \int_{t_0}^t\! X(s, F(s, \bm x))\,\dd{s}}\) となる。

      • アイディア

        • \({I_{\eps_0} \coloneqq (t_0 - \eps_0, t_0 + \eps_0),\quad \mathcal C = C^0(I_{\eps_0}, U).}\)

          \begin{align*} \varGamma[F(t, \bm x)] \coloneqq \bm x + \int_{t_0}^t X(s, F(s, \bm x))\,\dd{s} \in C^0(I_{\eps_0}, U). \end{align*}

          このとき \({F \in \mathcal C \implies \varGamma[F] \in C(I_{\eps_0} \times K, \RR^n).}\)

        • \({C^0(I_{\eps_0} \times K, \RR^n)}\) 上に次の距離を入れてリプシッツ性を利用する。

          \[\norm{F_1 - F_2} = \sup_{I \times K} \norm{F_1 - F_2}\]

      以下、上限記号の範囲指定を略記するが、だいたい \({I_{\eps_0} \times K}\) とかだ。

      • \({F_1, F_2 \in \mathcal C, \sup \norm{\varGamma[F_1] - \varGamma[F_2]} \le \eps_0 L \sup \norm{F_1 - F_2}}\) となる。

      • \({F_0(t, \bm x) = \bm x,\ F_1 = \varGamma[F_0]}\) とすると \({\sup \norm{F_1 - F_0} \le \eps_0 M.}\) ここで \({\eps_0 = \min\Set{\dfrac{1}{2L}, \dfrac{\eps}{4M}}}\) とすればよいことわがかる。

      • \({\eps_0 \le \dfrac{1}{2L}}\) と \({\eps_0 \le \dfrac{\eps}{4M}}\) から解の一意性と微分可能性をそれぞれ示す。

      • 微分可能性：

        • \({\norm{F_1 - F_0} \le \dfrac{\eps}{4}}\) より \({F_1 \in \mathcal C.}\)

        • \({F_{k + 1} = \varGamma[F_k] (F_k \in \mathcal C)}\) とすると次のように評価できる：

          \[\begin{split}\begin{align*} \norm{F_{k + 1} - F_k} &= \norm{\varGamma[F_k] - \varGamma[F_{k - 1}]}\\ &\le \frac{1}{2} \norm{F_k - F_{k - 1}}\\ &\le \frac{1}{2^k} \norm{F_1 - F_0}\\ &\le \frac{\eps}{2^{k + 2}}\\ \therefore \norm{F_{k + 1} - F_0} &\le \sum_{j = 0}^k \norm{F_{j + 1} - F_j} \le \sum{j = 0}^k \frac{\eps}{2^{j + 2}} \lt \frac{\eps}{2}. \end{align*}\end{split}\]

          ゆえに \(\set{F_k}\) は一様収束する。

        • 極限を \({F_\infty \in C^0(I_{\eps_0} \times K, \RR^n)}\) とすると \({F_\infty = \varGamma[F_infty].}\)

          \[\therefore F_\infty(t, \bm x) = \bm x + \int_{t_0}^t X(s, F_\infty(s, \bm x))\,\dd{s}.\]

      • 解の一意性：

        • \({F_p = \varGamma[F_p],\ F_q = \varGamma[F_q]}\) であるとすると、 \({\norm{F_p - F_q} \le \dfrac{1}{2}\norm{F_p - F_q}}\) ゆえに \({F_p = F_q.}\)

        • リプシッツ連続が関係する命題でいつもやっている証明方法だ。

  • 注意 6.3.2 によると仮定を少し緩められるらしい。\({(a, b) \times U \times \Lambda,\ \norm{X(t, \bm x_1, \lambda) - X(t, \bm x_2, \lambda)} \le L\norm{\bm x_1 - \bm x_2}.}\)

  • 問題 6.3.3: \(C^1\) 級ベクトル場と線形常微分方程式

    • ベクトル場 \({X(t, \bm x)}\) は \(t\) について \(C^1\) 級であり、

    • 解 \({F(t, \bm x) = (f_i(t, \bm x))_i}\) が次を満たすとする：

      • \({t = t_0}\) で初期値が \(\bm x\) である。

      • \({A_{ij}(t, \bm x) = \dfrac{\partial f_i}{\partial x_j}(t, \bm x)}\) （☆）が次を満たす：

        \[\diff{A_{ij}(t, \bm x)}{t} = \sum_{k = 1}^n \frac{\partial \xi_i}{\partial x_k}(t, F(t, \bm x)) A_{kj}(t, \bm x).\]

    このとき \(F\) は \(\bm x\) についても \(C^1\) 級である。

    • 証明の前半では \(F\) が偏微分可能であることを示す。つまり、\({F(t, \bm x)}\) の初期値に対する \(\bm v\) 方向の微分」が存在することを示す。そのために次の極限がある常微分方程式の解（で初期値が \(\bm v\) であるもの）であることをアダマールの補題を応用して示す：

      \[\lim_{s \to 0}\frac{F(t, \bm x + s\bm v) - F(t, \bm x)}{s}.\]

    • アダマールの補題より、次のように書ける：

      \[\begin{split}\begin{align*} &X(t + \bm x + \bm v) - X(t + \bm x) = \sum_{i = 1}^n v_i Y_i(t, \bm x, \bm v),\\ &Y_i(t, \bm x, \bm 0) = \frac{\partial X}{\partial x_i}(t, \bm x, \bm 0). \end{align*}\end{split}\]

      これを仮定の常微分方程式に当てはめると次のようになる：

      \[\begin{split}\begin{align*} &\diff{}{t}(F(t, \bm x + s\bm v) - F(t, \bm x)) = X(t, F(t, \bm x + s\bm v)) - X(t, F(t, \bm x))\\ &= \sum_{i = 1}^n(f_i(t, \bm x + s\bm v) - f_i(t, \bm x)) Y_i(t, F(t, \bm x), F(t, \bm x + s\bm v) - F(t, \bm x)). \end{align*}\end{split}\]

    • 行列 \({Y = (Y_i)_i}\) に対して \({Y(s) = (Y_i(t, F(t, \bm x), F(t, \bm x + s\bm v) - F(t, \bm x))_i}\) とおくと、常微分方程式 \({\displaystyle \diff{\bm u}{t} = Y(s)\bm u}\) （★）の解は先ほどの極限の対象となる。

      • \(Y(s)\) は \(s\) について連続である。

      • そして \({s = 0}\) のときも連続に定義されている。解は（★）の解に収束する。この根拠は 注意 6.3.2 から来ている。

        \[Y(0) = \frac{\partial \xi_i}{\partial x_j}(t, F(t, \bm x))\]

    • （☆）の \({A_{ij}(t, \bm x)}\) は次を満たす：

      • \({A_{ij}(t_0, \bm x) = \delta_{ij}}\)

      • （☆）の次に書いた常微分方程式

      \({\displaystyle \frac{\partial \xi_i}{\partial x_k}(t, F(t, \bm x))}\) を \(\bm x\) をパラメーターとして見ると再び 注意 6.3.2 から \(A_{ij}\) は \(\bm x\) について連続である。

      したがって \(F\) は \(\bm x\) について \(C^1\) 級である。

  • 注意 6.3.4: \({\displaystyle \frac{\partial F(t, \bm x)}{\partial t} = X(t, F(t, \bm x))}\) は \(\bm x\) と \(t\) のどちらについても連続であることが \(F\) が \(C^1\) 級であることに注意。

  • 問題 6.3.5: ベクトル場が \(C^\infty\) 級ならば解は \(C^\infty\) 級である

    \(\fn{X}{(a, b) \times U}\RR^n\) が \(C^\infty\) 級であるならば、 \({t = t_0}\) で解 \(\bm x\) を持つ関数 \({F(t, \bm x)}\) は \(C^\infty\) 級である。

    • まず \({A_{ij}(t, \bm x) = \dfrac{\partial f_i}{\partial x_j}(t, \bm x)}\) とおくとこうなる（再掲）：

      \[\diff{A_{ij}(t, \bm x)}{t} = \sum_{k = 1}^n \frac{\partial \xi_i}{\partial x_k}(t, F(t, \bm x)) A_{kj}(t, \bm x).\]

    • 帰納法で示す：「\(X\) が \(C^r\) 級であるとき、解 \(F\) も \(C^r\) 級である」とすると、

      1. 偏微分の項は \(C^r\) 級である。なぜならば \(\dfrac{\partial \xi_i}{\partial x_k}\) と \(F\) の両方が \(C^r\) 級であるから。

      2. \({\displaystyle \diff{F(t, \bm x)}{t} = X(t, F(t, \bm x))}\) については \(X\) と \(F\) が \(C^r\) 級だから \(t\) について \(C^r\) 級である。

      ゆえに \(F\) は \(C^{r + 1}\) 級である。

  • 注意 6.3.6

    1. \(C^\infty\) 級ベクトル場フローを生成するならば、そのフローは \(C^\infty\) 級である。

    2. パラメーターに依存する場合、パラメーターに対して \(C^\infty\) 級に依存するベクトル場が生成するフローもまたパラメーターに対して \(C^\infty\) 級に依存する。

6.4 コンパクト多様体上のベクトル場¶

コンパクト多様体上の \(C^\infty\) 級ベクトル場はフロー \(F_t\) を生成する。

  • 定理 6.4.1: \({\displaystyle X = \diff{F_t}{t} \circ F_{-t}}\)

    • いつもの開被覆からさらに次のような \(V_i, W_i\) をとる：

      • \({U_i \supset \closure{V_i} \supset V_i \supset \closure{W_i} \supset W_i}\)

      • \({\bigcup W_i = M}\)

    • 証明は三段階に分けて理解する。

      1. \(C^\infty\) 級写像 \({\displaystyle \fn{F}{(-\eps, \eps) \times M}\RR^n,\quad \diff{F}{t}(t, x) = X(F(t, x))}\) が存在することを示す。

        • \({\displaystyle X = \sum_{k}\xi_k^{(i)}\frac{\partial}{\partial x_k^{(i)}}}\) とおく。閉包のコンパクト性から \({\varphi_i(V_i)}\) 上で有界連続でリプシッツ条件を満たす。

        • \({\varphi_i(\closure{W_i}) \subset \varphi_i(V_i)}\) において次のような \({\eps^{(i)} \gt 0}\) が存在する：

          \begin{align*} & \fn{F^{(i)}}{(-\eps^{(i)}, \eps^{(i)}) \times \varphi_i(\closure{W_i})}\varphi_i(V_i)\\ & \diff{F^{(i)}}{t}(t, \bm x) = \xi^{(i)}(F^{(i)}(t, \bm x))\\ \end{align*}

          ここで \({\xi^{(i)} = (\xi_1^{(i)}, \dotsc, \xi_n^{(i)})}\) とした。

        • \({\eps = \min\set{\eps^{(i)}}}\) とすると全ての \({x \in M}\) に対して次の性質を満たすような近傍 \(W^i\) が存在する：

          \begin{align*} & F^i(t, x) = \varphi_i\inv(F^{(i)}(t, \varphi_i(x)))\\ & \fn{F_x^i}{(-\eps, \eps)}M\\ & \diff{F_x^i}{t}(t) = X(F_x^i(t)). \end{align*}

        • 別の \({W_j \owns x}\) について \(F_x^j\) を上のように定義すると \({F_x^j = F_x^i}\) となる（常微分方程式の解の比較）。

      2. 解の接続。上記写像の定義域を \({\RR \times M}\) にまで拡張した写像が存在することを示す。

        • \({\displaystyle t \in \left(-\eps,\ \frac{\eps}{2}\eps\right)}\) のときは \({\displaystyle t \in \left(\frac{n - 1}{2}\eps,\ \frac{n + 1}{2}\eps\right)}\) に対して改めて \({\displaystyle F(t, x) = F\!\left(t - \frac{n - 1}{2}\eps,\ F\!\left(\frac{n - 1}{2}\eps,\ x\right)\!\right)}\) とする。

        • \({\displaystyle t \in \left(-\frac{\eps}{2}\eps,\ \eps\right)}\) のときは上でやった区間の「反対側」に対して改めて \({\displaystyle F(t, x) = F\!\left( t + \frac{n - 1}{2}\eps,\ F\!\left( -\frac{n - 1}{2}\eps,\ x\right)\!\right)}\) とする。

        どちらも \({\displaystyle \diff{F}{t}(t, x) = X(F(t, x))}\) が成り立っている。

      3. \({F(t + s, x) = F(t, F(s, x))}\) を示す。

  • 例題 6.4.3: 連結コンパクト一次元多様体が向き付け可能であることを使えば、多様体上にゼロでないベクトル場が存在することがわかり、\(M\) と \({\RR/\ZZ}\) は微分同相であると言える。

    • 各座標近傍 \(U_i\) に対し 1 の分割 \(\lambda_i\) を導入する。

    • \(U_i\) に台を持つベクトル場 \({\displaystyle \lambda_i \frac{\partial}{\partial t^{(i)}}}\) を考える。ここで \(t^{(i)}\) は座標とする。

    • \({\displaystyle X = \sum_i \lambda_i \frac{\partial}{\partial t^{(i)}} \ne 0}\) が求めるものの一つである。

      • \({D\gamma_{ij} \gt 0}\) および \({\sum_i \lambda_i = 1}\) を利用する。

        \[X = \sum_i \lambda_i (D\gamma_{ji} \circ \varphi_i)\frac{\partial}{\partial t^{(i)}} \gt 0.\]

    • \(\varphi_t\) を \(X\) のフローとする（記号が座標とカブっている）。フローの軌道は前に述べたように 3 通りなので、これで場合分けする。

      1. 一点のみのケースはあらかじめ仮定より除外されている。

      2. \(M\) 全体となるケースはコンパクト性により円周と同相である。

      3. \(M\) と一致しないケース：

        \({A = \set{\varphi_t(x_0) \sth t \in \RR}}\) をとり、\({x \in M \setminus A}\) とする。ここで次のような近傍 \({U_i \owns x}\) が存在するが、これは \(X\) の \(U_i\) 上の軌道が \(U_i\) を含むことになり、 \({U_i \subset A}\) となって \({x \in M \setminus A}\) に矛盾する：

        \[x \in U_i,\ U_i \cap A \ne \varnothing.\]

  • 注意 6.4.4 は長いが、連結コンパクト一次元多様体が向き付け可能であること自体の証明だ。フロー、連結成分直和分割、接写像、中間値の定理などを使う。

  • 問題 6.4.5: 次の条件を満たす \({\displaystyle \mu\frac{\partial}{\partial x_1}}\) が生成するフロー \(\varPhi_t\) の \({\displaystyle \lim_{t \to \infty}\varPhi_t(\bm x)}\) と \({\displaystyle \lim_{t \to -\infty}\varPhi_t(\bm x)}\) の値

    • \(\fn{\mu}{\RR^n}\RR\) は \(C^\infty\) 級で、

    • \({\supp \mu = \set{\bm x \in \RR^n \sth \norm{\bm x} \le 1},}\)

    • \({\norm{\bm x} \lt 1 \implies \mu(\bm x) \gt 0}\) と仮定する。

    1. 単位超球面上およびその外側

      \({\mu(\bm x) = 0}\) なので極限値はどちらも \(\bm x\) となる。

    2. 単位超球面内部の場合

      • 内部ではベクトル場が非ゼロであるので、どちらの極限も球面上の点となる。

      • \({\varPhi_t(\bm x) = (\varphi_t(\bm x), x_2, \dotsc, x_n)}\) と書けるが、

        • \({t \lt 0 \implies \varphi_t(\bm x) \lt x_1,}\)

        • \({t \gt 0 \implies \varphi_t(\bm x) \gt x_1}\)

        なので次のようになるという解釈か：

        \[\lim_{t \to \pm\infty}\varPhi_t(\bm x) = ( \pm\sqrt{1 - x_2^2 - \dotsb - x_n^2}, x_2, \dotsc, x_n).\]

      解答例では具体的に \(\varphi_t\) を求めていないように読める。

6.5 連結多様体上の部分集合の比較¶

連結多様体上の点は平等であり、特別な点は存在しない (p. 135)。

  • 例題 6.5.1: 連結多様体上の任意の二点について、一方を他方に写す微分同相写像が存在する。

    • 証明には 問題 6.4.5 の技法を用いる。

    • 連結多様体上の一点を固定して、微分同相で写り合う点は同値関係となる。この同値類が開集合であることを示す必要がある。

      \[A_{x_1} = \set{x \in M \sth \exists \fn{F}{M}M,\ F(x_1) = x}.\]

      を考えると、こういう集合が同値類になって \(M\) を非交和分割できる。

    • <同値類が開集合であることを示したから、閉集合でもある> (p. 135) とあるが、ここが理解できない。

  • 注意 6.5.2

    • アイソトピーもとれることが上のようにしてわかる。

    • コンパクト外では恒等写像としておけば微分同相写像であるようにできる。

  • 問題 6.5.3: 多様体次元が 2 以上の連結多様体上には、相異なる \(n\) 点と別の相異なる \(n\) 点とについて、一方を他方に写す微分同相写像が存在する。

    • 証明は帰納法による。

      1. 一点対一点の場合は 例題 6.5.1 により成り立つ。

      2. \(x_1, \dots, x_{n - 1} \in M\) を \(y_1, \dots, y_{n - 1} \in M\) にそれぞれ写す微分同相写像 \(\fn{F_1}{M}M\) が存在すると仮定する。

        • \({M' = M \setminus \set{y_1, \dots, y_{n - 1}}}\) とおくと、これは連結である（この証明がメインか？）。

        • \({\fn{F'}{M'}M,\quad F'(F_1(x_n)) = y_n}\) となる \(F'\) が 例題 6.5.1 により存在する。

        • \(F'\) の定義域を元の多様体に次のように拡張して \(\hat{F'}\) とすれば、\({F = \hat{F'} \circ F_1}\) が求める微分同相写像だ。

          • 点 \(y_i\) の近傍の点 \(y\) に対しては \({F' = \id\ \text{i.e. } F'(y) = y.}\)

  • 問題 6.5.4: 連結多様体上の任意の二点 \(x_0, x_1\) に対して次のようなフロー \(F_t\) が存在する： \({F_t(x_0) = x_1.}\)

    • 問題 6.4.5 の要領で座標近傍に台を持つフロー \(F_t\) を構成する。

    • 問題 6.5.3 によると次のような微分同相写像 \(\fn{G}{M}M\) が存在する： \({G(y_0) = x_0,\ G(y_1) = x_1.}\)

    • 合成写像 \({G \circ F_t \circ G\inv}\) を改めて \(F_t\) とすればそれが求めるフローである。

\(\fn{F}{M}N\) の二つの正則値 \(y_0, y_1\) に対する \({F\inv(y_0), F\inv(y_1)}\) の比較をしたい。

  • 例題 6.5.5: ベクトル場の射影

    • \(M, N\) をコンパクト連結多様体、

    • \({F \in C^\infty(M, N),}\)

    • \(\xi, \eta\) をベクトル場とする。

    • \(\xi, \eta\) の生成するフローをそれぞれ \(\varphi_t, \psi_t\) とする。

    • \({\forall x \in M, F_*(\xi(x)) = \eta(F(x)) \implies F(\varphi_t(x)) = \psi_t(F(x)).}\)

    • 証明するには \({\displaystyle \diff{F(\varphi_t(x))}{t} = \eta(F(\varphi_t(x)))}\) を示す。

      \[\diff{F(\varphi_t(x))}{t} = F_*\left(\diff{\varphi_t(x)}{t}\right) = F_*\left(\xi(\varphi_t(x))\right) = \eta(F(\varphi_t(x))).\]

      等号の根拠は順に接写像の性質、ベクトル場 \(\xi\) がフロー \(\varphi_t(x)\) を生成すること、射影条件による。

      一方、\({t = t_0}\) のとき \({F(\varphi_0(x)) = F(x).}\) 常微分方程式の解の一意性により \({\psi_t(F(x)) = F(\varphi_t(x)).}\)

  • 問題 6.5.6: コンパクト連結多様体 \(M\) 上の \(C^\infty\) 級関数 \(f\) について \({[a, b] \subset \RR}\) はすべて正則値であるとすると、次が成り立つ： \({f\inv([a, b])}\) と \({f\inv(a) \times [a, b]}\) は微分同相である。

    • 次のようにおく：

      • \(\set{(U_i, \varphi_i)}_{i = 1, \dotsc, k}\) を有限開被覆、

      • \({\lambda_i(x) \le 0}\) を \(U_i\) を台とする 1 の分割、

      • また、\({f\inv([a, b])}\) と交わる \(\varphi_t\) の第一座標成分 \({x_1{(i)} = f}\) であると仮定してもよい（なぜか）。

    • ベクトル場 \({X_i = \lambda_i(x)\dfrac{\partial}{\partial x_1^{(i)}}}\) は \({M \setminus U_i}\) 上ゼロである。

    • \({(f_*)_x(X_i) = \lambda_i(x)\dfrac{\partial}{\partial t}.}\)

    • \({\xi = \sum(X_i)}\) とすると \({(f_*)_x\xi(x) = \sum\lambda_i(x)\dfrac{\partial}{\partial t} = \dfrac{\partial}{\partial t}.}\) したがって \(\xi\) の生成するフロー \(F_t\) について次が成り立つ：

      \[x \in f\inv([a, b]),\ a - f(x) \le t \le b - f(x) \implies f(F_t(x)) = f(x) + t.\]

    • 次に写像 \({\fn{p}{f\inv([a, b])}f\inv(a)}\) を \({x \longmapsto F_{a - f(x)}(x)}\) で定義する（なかなか思いつかない）。

      • さらに写像 \({\fn{(p, f)}{f\inv([a, b])}f\inv \times [a, b]}\) を考える。この逆写像を \({G(x', t)}\) とおくと \({G(x', t) = F_{t - a}(x').}\)

      • \({(p, f)}\) と \(G\) のどちらも \(C^\infty\) 級であるので、これらは微分同相写像である。

  • 問題 6.5.7: フローボックス定理

    • 仮定

      • \(M\) を \(m\) 次元コンパクト多様体、

      • \({N \subset M}\) を \({m - 1}\) 次元コンパクト部分多様体、

      • \(\xi\) をベクトル場、

      • \(\varphi_t(x)\) を \(\xi\) が生成するフロー、

      • \({\forall x \in N, \xi(x) \notin T_x N}\) とする（部分多様体から見れば接ベクトルがはみ出す）。

    • 結論

      • 次を満たす写像と数 \({\eps \gt 0}\) が存在する：写像 \({(-\eps, \eps) \times N \longto M}\) が \(M\) の開集合への埋め込みである。

    • 証明

      • まず写像 \(\fn{F}{\RR \times N}M\) を \({F: (t, x) \longmapsto \varphi_t(x)}\) で定義する。\({t = 0}\) で接写像 \(\fn{F_*}{\RR \times T_xN}T_xM\) を考えると点 \({(0, x)}\) で次が成り立つ：

        \[\begin{split}\begin{align*} &F_*|T_xN = \id_{T_xN},\\ &F_*|\left(\frac{\partial}{\partial t}\right)_{(0, x)} = \xi(x). \end{align*}\end{split}\]

        よって \({\rank F_* = m}\) である。

    • 逆写像定理より \({\zeroset \times N}\) 上では \(F\) は単射である。

    • 例題 4.3.1 より \(F\) は

      • \({\zeroset \times N}\) の近傍、すなわち \({(-\eps, \eps) \times N}\) から

      • \({N \subset M}\) の近傍への

      微分同相写像である。
