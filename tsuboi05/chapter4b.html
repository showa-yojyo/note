4.4 部分多様体¶

  • 定義 4.4.1: 部分多様体

    • 多様体 \(N\) に対して \(M \subset N\) に次の性質があるとき、それを p 次元部分多様体であるという。

      \begin{align*} \forall x_0 \in M, \exists(U, \varphi): M \cap U = \set{x \in U \sth x_{p + 1} = \dots = x_n = 0} \end{align*}
微分可能構造
座標近傍系の同値類について注意しておく。座標近傍系の両立という概念の上位互換な概念が微分可能構造だ。

\(\fn{F}{M}N,\ \rank F_* = \min\set{m = \dim M, n = \dim N}\) とすると、

\[\begin{split}(y_i \circ F \circ \varphi\inv) = \begin{cases} x_i & \quad \text{if } m \le n \text{ and } i = 1, \dotsc, m\\ 0 & \quad \text{if } m \le n \text{ and } i = m + 1, \dotsc, n\\ x_i & \quad \text{if } m \ge n \text{ and } i = 1, \dotsc, n. \end{cases}\end{split}\]

最後のケースでは特に \(F\inv(F(x_0)) \cap U = \set{x \in U \sth x_{m + 1} = \dots = x_n = 0}\) は \(m - n\) 次元部分多様体である。

はめ込み
\(m < n\) のときに、写像 \(\fn{F}{M}N\) に対し、\(\rank F_* = m\) であればはめ込みであるという。
埋め込み

はめ込み \(F\) によって \(N\) の位相から誘導される位相が \(M\) の位相そのものと一致するような \(F\) は埋め込みであるという。

  • \(F(M)\) は \(N\) の正則な部分多様体となる。
  • 定理 4.4.2: はめ込みが単射であれば、コンパクトな多様体の像は部分多様体となる。
沈み込み

\(m \ge n\) の場合に任意の \(x \in M\) に対して \(\rank F_* = n\) となるとき、この写像は沈み込みであるという。

  • \(F\inv(y)\) は \(m - n\) 次元部分多様体となる。
  • この特殊な状況を 例題 8.6.1 ファイブレーション定理 で取り扱う。
  • 例題 4.4.3: 部分多様体間の \(C^\infty\) 級写像が誘導する写像も \(C^\infty\) 級となる。
    • 仮定をまとめる：
      • \(N_1, N_2\) をそれぞれ \(n_1, n_2\) 次元多様体とし、
      • \(C^\infty\) 級写像 \(\fn{F}{N_1}N_2\) があり、
      • \(M_1 \subset N_1,\ M_2 \subset N_2\) はそれぞれ部分多様体であり、
      • さらに \(F(M_1) \subset M_2\) が成り立つとする。
    • \(M_2 \subset N_2\) が部分多様体であるので、次のような \(F(x_0) \in M_2 \subset N_2\) の座標近傍 \((V, \varphi),\ \varphi = (y_1, \dotsc, y_{n_2})\) がある： \(M_2 \cap V = \set{y_{{m_2} + 1}, \dotsc, y_{n_2} = 0}.\)
    • 残りの座標成分のほうを考えと \((y_1, \dotsc, y_{m_2}) \circ F \circ \varphi\inv\) が \(C^\infty\) 級となる。したがって 誘導される写像 \(\fn{G}{M_1}M_2\) も \(C^\infty\) 級となる。
  • 問題 4.4.4: \(GL_2(\RR)\) と \(SL_n(\RR)\)
    • 前者は \(n^2\) 次元多様体、後者は \(n^2 - 1\) 次元部分多様体である。
      • 大前提として \(M_n(\RR)\) は座標近傍系を空間全体を近傍とし、各成分の値をそのまま座標とする座標近傍一つからなるものを考えれば、これは \(n^2\) 次元多様体である。
      • GL については \(GL_n(\RR) = \set{A \in M_n(\RR) \sth \det A \ne 0}\) である。連続写像 \(\fn{\det}{M_n(\RR)}\RR\) の開集合 \(\set{x \in \RR \sth x \ne 0}\) の逆像とみなすことで開集合となり、先ほどと同じ要領で座標近傍系を構成すれば \(n^2\) 次元多様体である。
      • SL について。まず定義を書き下してみると \(SL_n(\RR) = \set{A \in GL_n(\RR) \sth \det A = 1}\) だ。今度は \(\det\inv(1)\) である。
        • \(\det\) のヤコビ行列なるものを考える。余因子展開を意識することで \(\det(x_{ij}) = \sum x_{ij} A_{ij}\) のように書かれるから、 \(\displaystyle \frac{\partial \det}{\partial x_{ij}} = A_{ij}\) で、右辺は \((n - 1)^2\) 次正方行列の行列式になっている。
        • SL では \(D\det \ne 0\) であることに注意。 \(\det\) は \(n^2\) 次元から \(1\) 次元への関数であり、陰関数定理により、ある近傍 \(W\) と \(C^\infty\) 級写像（座標になる） \(\fn{\varphi}{W}\RR^{n^2 - 1}\) が存在する。
    • 行列の積、逆行列を取る演算はどちらも \(C^\infty\) 級の写像である。
      • 両者ともリー群なので 問題 4.3.3 を利用できる。
      • 積も逆行列も多項式の演算に、せいぜい非ゼロの値の除算が加わるものなので \(C^\infty\) 級の写像だという主張には問題あるまい。

  • 問題 4.4.5: 直交群 \(O(n)\) は多様体である。

    • 写像 \(C: A \longmapsto {}^t\!AA\) を考える。

      • 直交群は \(C\inv(I_n)\) となる。
      • \(C\) は \(C^\infty\) 級である。

    • 微分を考える。ここが少し思いつかない。

      • まずは \(C(A + X) - C(A) = {}^t\!XA + {}^t\!AX + {}^t\!XX.\) これの極限 \(X \to O\) が \(DC_{(A)}X\) である。

      • \(DC_{(A)}X = {}^t\!XA + {}^t\!AX\) という形をよく見ると二つの写像 \(X \longmapsto AX\) と \(X \longmapsto {}^t\!X + X\) の合成であることに気付く。

        • \(\rank(X \longmapsto AX) = n^2\) である。
        • \(\displaystyle \rank({}^t\!X + X) = \frac{n(n + 1)}{2}\) である。

        以上より合成写像のランクは \(\displaystyle \frac{n(n + 1)}{2}\) である。

    • よって陰関数定理により、\(C\inv(I_n) = O(n)\) の近傍 \(W\) と \(C^\infty\) 級写像 \(W \longto \RR^\frac{n(n - 1)}{2}\) が存在する。

      • 写像 \(\fn{C}{M_n(\RR)}M_n(\RR)\) は \(n^2\) 次元空間から \(\displaystyle \frac{n(n + 1)}{2}\) 次元空間へのものだとわかったことによる。

  • 例題 4.4.7: 横断的に交わる二つの部分多様体の共通部もまた部分多様体である。

    • \(\forall x \in X \cap Z, T_x Y + T_x Z = T_x X\) が仮定、 \(Y \cap Z \subset X\) が部分多様体であることが結論。

    以下、ノートの都合上 \(a = \dim X - \dim Y,\ b = \dim X - \dim Z\) と書く。

    • \(x_0 \in Y \cap Z\) の座標近傍を \((U, \varphi)\) とする。

    • まず次のような写像 \(\fn{F_Y}{U}\RR^a\) が存在する： \(U \cap Y = F_Y\inv(F_Y(x_0)).\)

    • さらに \(\forall x \in U, \rank F_{Y*} = a\quad(\fn{F_{Y*}}{T_xX}\RR^a.)\)

    • 同様な性質の \(F_Z\) も存在する。

    • 写像の直積を構成することで \(F_{Y*}, F_{Z*}\) を適当に制限すると同型写像が得られることを示す：

      \[\fn{(F_{Y*}, F_{Z*})}{U}\RR^a \times \RR^b.\]
      • 接空間を \(T_xY = (T_xY \cap T_xZ) \oplus V_Y\) のように部分空間の直和に分解する。このとき \(F_{Y*}|V_Y \cong \RR^a\) となる。 \(T_xZ\) についても同様に \(V_Z\) を定義する。
      • \((F_{Y*}, F_{Z*})|(V_Y \oplus V_Z) \cong \RR^a \oplus \RR^b.\) \(\rank(F_{Y*}, F_{Z*}) = a + b = 2\dim X - \dim Y - \dim Z.\) が得られた。

    • \(U \cap (Y \cup Z) = (F_Y, F_Z)\inv(F_Y(x_0), F_Z(x_0))\) において、 \(X\) 内で \(U\) を動かすことによって \(Y \cap Z\) が \(\dim X - (2\dim X - \dim Y - \dim Z) = \dim Y + \dim Z - \dim X\) 次元の部分多様体であることがわかる。

4.5 接束（展開）¶

冒頭のユークリッド空間内の多様体から多様体と接空間のペアの空間を構成する部分は前座。

接束

前章の記号 \(V_i, V_{ij}, \gamma_{ij}\) 等を流用する。次のようにして構成する商空間を多様体の接束という：

  1. 直和 \(\bigsqcup V_i\) に同値関係 \(x_i \sim x_j \iff x_i = \gamma_{ij}(x_j)\) を導入する。このとき、商空間 \(X = (\bigsqcup V_i / \sim)\) は \(M\) と微分同相になる（ 例題 3.5.2 などを参照）。

  2. 直積の直和 \(\bigsqcup (V_i \times \RR^n)\) に次の同値関係を導入する。

    \begin{align*} (x_i, v_i) \sim (x_j, v_j) \iff \exists \gamma_{ij}: x_i = \gamma_{ij}(x_j),\ v_i = (D\gamma_{ij})_{(x_j)} v_j \end{align*}

    同値関係となる理由：

    • 写像 \(G_{ij}: (x_i, v_i) \longmapsto (\gamma_{ij}(xj), (D\gamma_{ij})_{(x_j)} v_j)\) を考える。これは微分同相となる。
    • そして \(G_{ij} \circ G_{jk} = G_{ik}\) （ただし \(G_{ii} = \id\) と約束する）が成り立つ。

    このとき、商空間 \(Y = (\bigsqcup (V_i \times \RR^n))/\sim\) はハウスドルフとなり、 \(2n\) 次元多様体となる。

    ハウスドルフとなる理由（面倒）：

    • 射影をいくつか定義して、その合成写像による商空間の開集合の逆像もまた開集合であることを示し、 \(Y \longto X\) に連続写像が存在することを示せる。
    • 次に、直和から商空間への射影二種 \(p_x, p_y\) を適宜制限して同相写像を得る。
    • 写像 \(\fn{P\inv}{p_x(V_i)}p_x(V_i) \times \RR^n\) が同相であることを示す。
    • 最後に 問題 3.5.3 を利用する。

接束は ベクトル束 の一種である (pp. 85-86)。

  • 問題 4.5.2: \(F \in C^\infty(M, N)\) の引き起こす接束の間の写像 \(\fn{F_*}{TM}TN\) は \(C^\infty\) 級である。

    • \(TM\) は \(\bigcup \varphi_i(U_i) \times \RR^m\) の商空間である。

    • \(TM\) の座標近傍系は \(\varphi_i(U_i) \times \RR^m\) の像およびそこからの逆写像として定義される。

    • 以上、\(TN\) も同様。

    • それゆえ、次の写像は \(F\) が \(C^\infty\) 級であれば \(C^\infty\) 級となる：

      \[(\bm u_i, \bm v_i) \longmapsto ((\psi \circ F \circ \varphi_i\inv)(\bm u_i), D(\psi \circ F \circ \varphi_i\inv)_{(\bm u_i)}(\bm v_i)).\]

  • 問題 4.5.3: ユークリッド空間内の多様体 \(M \subset \RR^N\) に対して \(TM\) と \(X = \set{(x, v) \sth x \in M, v \in T_x M}\) は微分同相である。

    • 点 \(\bm x^0 \in M\) の近傍におけるグラフ表示から \(X\) のグラフ表示を構成する。

      \[\bm x^0 = (\bm x_1^0, \bm x_2^0) \in \RR^p \times \RR^{N - p},\]

      \(\bm x_1^0 \in W\) 上のグラフ表示 \(\fn{g}{W}\RR^{N - p},\ M \cap U = \set{(\bm x_1, g(\bm x_1)) \sth \bm x_1 \in W}\) とすると \(\bm v \in T_{(\bm x_1, g(\bm x_1))}M \iff \bm v = (\bm v_1, Dg_{(\bm x_1)}\bm v_1)).\)

      \(X \cap (U \times \RR^N) = \set{(\bm x_1, g(\bm x_1)), (\bm v_1, Dg_{(\bm x_1)}\bm v_1)) \sth (\bm u_1, \bm v_1) \in W \times \RR^p}.\)

    • \(TM\) は \(M\) のパラメーター表示による被覆 \(\set{\Phi_i(W_i)}\) について商空間 \(\bigcup W_i \times \RR^p\) として定義されている。

    • 写像 \((\bm u, \bm v) \longmapsto (\Phi_i(\bm u), {D\Phi_i}_{(\bm u)}(\bm v))\) は連続であり、 \(TM\) の定義から逆写像もまた連続、\(C^\infty\) 級である。
