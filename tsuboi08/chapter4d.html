4.4 リーマン多様体上の微分形式とベクトル場¶

  • リーマン計量 \(\fn{g_x}{T_xM \times T_xM}\RR\) は余接空間のテンソル積の元である：

    \[g_x \in T_x^*M \otimes T_x^*M.\]

  • 座標近傍の変換：

    \[g = \sum_{i, j}^n g_{ij}(\bm x)\dd x_i \otimes \dd x_j = \sum_{k, l}^n \sum_{i, j}^n g_{ij}(\bm x(\bm y)) \frac{\partial x_i}{\partial y_k} \frac{\partial x_j}{\partial y_l} \dd y_k \otimes \dd y_l\]

    を見ると、リーマン計量 \(g\) はベクトル束 \({(T^*M \otimes T^*M, M, p)}\) における射影 \(\fn{p}{T^*M \times T^*M}M\) に対し \({p \circ g = id_M}\) になるような切断である。細かく言うと、対称なテンソル積のなす \(\dfrac{n(n + 1)}{2}\) 次元ベクトル空間をファイバーとする部分ベクトル束の切断である。

  • リーマン計量があると、ベクトル場と 1 形式の間に全単射が存在する：

    \[\alpha(v) = g_x(v, \xi(x))\quad\forall v \in T_xM.\]

    • \(g_x\) が双線型かつ非退化であることに注意。

  • 逆に 1 形式 \(\alpha\) に対して上の式を満たすベクトル場 xi が一意的に定まる。

    • \(\displaystyle {\alpha_i = \sum_{j = 1}^n g_{ij} \xi_j}\) から \(\displaystyle {\xi_j = \sum_{i = 1}^n g^{ij}\alpha_i}\) とすればよい。

    • 座標近傍上では常に \(\displaystyle {\sum_i^n\alpha_i v_i = \sum_{i, j}^n g_{ij}v_i\xi_j}\) である。

  • 逆行列 \(g^{ij}\) に関しては双線型 \(\fn{\overline g_x}{T_x^*M \times T_x^M}\RR\) が次で定まる：

    \[\overline g_x(\sum_{i = 1}^n \alpha_i\,\dd x_i, \sum{j = 1}^n\beta_j\,\dd x_j) = \sum_{i, j = 1}^n g^{ij}\alpha_i\beta_j.\]

    さらに \(\alpha, \beta\) がベクトル場 \(\xi, \eta\) にそれぞれ対応するときは：

    \[\begin{split}\begin{align*} \overline g_x(\alpha, \beta) &= \sum_{i, j = 1}^n g^{ij}\alpha_i\beta_j\\ &= \sum_{i, j = 1}^n g^{ij} \sum_{k = 1}^n g_{ik}\xi_k \sum_{l = 1}^n g_{il}\eta_l\\ &= \sum_{k, l = 1}^n g_{kl}\xi_k\eta_l. \end{align*}\end{split}\]

  • 接空間 \(T_xM\) の正規直交基底

    \[\Set{\xi^{(k)} = \sum_{i = 1}^n \xi_i^{(k)}\frac{\partial}{\partial x_i}}\]

    をとる。正規直交基底なので \(g_x{(\xi^{(k)}, \xi^{(l)})} = \delta_{kl}\) となる。

    これに対応する余接空間 \(T_x^*M\) の基底

    \[\Set{\alpha^{(n)} = \sum_{i, j = 1}^n g_{ij}\xi_j^{(k)}\,\dd x_i}\]

    もまた正規直交基底となる。これは先述の座標近傍上における等式による。

  • 正規直交基底が存在することは、グラム・シュミットの直交化法により言える。「ベクトル」の大きさの調整にリーマン計量を用いる：

    \[\begin{split}\begin{align*} v_1 &= \frac{\partial}{\partial x_1},\\ \xi^{(1)} &= \frac{v_1}{\sqrt{g(v_1, v_1)}} \end{align*}\end{split}\]

    から開始して、\(\xi^{(1)}, \dotsc, \xi^{(k)}\) が定まる。それから：

    \[\begin{split}\begin{align*} v_{k + 1} &= \frac{\partial}{\partial x_{k + 1}} - \sum_{i = 1}^k g\left(\xi^{(i)}, \frac{\partial}{\partial x_{k + 1}}\right)\xi^{(i)},\\ \xi^{(k + 1)} &= \frac{v_{k + 1}}{\sqrt{g(v_{k + 1}, v_{k + 1})}} \end{align*}\end{split}\]

    と正規化する。

  • リーマン多様体の体積形式

    • 余接空間 \(T_x^*M\) の正規直交基底 \(\set{\alpha^{(k)}}\) に対して、 \(n\) 形式 \({\alpha^{(1)} \wedge \dotsb \wedge \alpha^{(n)}}\) は（符号を除いて）リーマン計量で定まる。

    • 多様体が向き付け可能であれば、各座標近傍上で正の正規直交基底がとれる。そのとき、上述の \(n\) 形式の外微分をとると、二つの座標近傍の共通部では座標変換のよる引き戻しとなる。このような非ゼロ微分形式をリーマン多様体の体積形式という。

  • 問題 4.4.1: 向き付けられたリーマン多様体の向き付けられた座標近傍における体積形式

    この問題でリーマン多様体の体積形式を具体的に求める。

    1. リーマン計量を \(g_{ij}\) とする：

      \[g = \sum g_{ij} \dd x_i \otimes \dd x_j.\]

    2. 余接空間 \(T_x^*M\) の正規直交基底を \(\alpha^{(k)}\) とする：

      \[\alpha^{(k)} = \Set{\sum_{i = 1}^n \alpha^{(k)}\,\dd x_i}.\]

    3. 本節冒頭で述べた双線型写像を適用すると \(\displaystyle {\sum_{i, j = 1}^n g^{ij}\alpha_i^{(k)}\alpha_i^{(l)} = \delta_{kl}}\) すなわち \({{}^t\!AG\inv A = I_n}\) の形に書ける。ここで \({G = (g_{ij}),} \ {A = (\alpha_i^{(k)})}\) とした。

    4. 問題 1.8.5 の結果より

      \[\alpha^{(1)} \wedge \dotsb \wedge \alpha^{(n)} = \det A \,\dd x_1 \wedge \dotsb \wedge \dd x_n.\]

    5. 3. より \({(\det A)^2 = \det G}\) である。

    6. 4. と 5. より求める体積形式は次のように表せる：

      \[\alpha^{(1)} \wedge \dotsb \wedge \alpha^{(n)} = \sqrt{\det G}\,\dd x_1 \wedge \dotsb \wedge \dd x_n.\]

    この右辺が向き付けられたリーマン多様体 \({(M, g)}\) の体積形式である。以下で \(\varOmega_{(M, g)}\) とあるのは、これである。

  • 定理 4.4.2: ガウス・グリーンの公式

    向き付けられたコンパクトリーマン多様体の体積形式に関するベクトル場の発散に関する公式だ。

    • \(\varOmega_{(M, g)}\) をリーマン多様体の体積形式とし、

    • \(\xi\) を \({L_\xi\varOmega_{(M, g)}} = {\div(\xi) \varOmega_{(M, g)}}\) を満たすベクトル場であるとし、

    • \(n\) は単位ベクトル場であり、多様体の境界 \({\partial M}\) においてそれに直交かつ外向きであるとする。

    とする。このとき、次の積分に関する等式が成り立つ：

    \[\int_M\!\div(\xi)\varOmega_{(M, g)} = \int_{\partial M}\!g(n, \xi)\varOmega_{(\partial M, g|\partial M)}.\]

    この積分は、境界がない多様体についてはゼロであると解釈する。

    1. ベクトル場 \(\xi\) についての仮定および 命題 4.1.8 カルタンの公式より左辺を次のように変形できる：

      \[\begin{split}\begin{align*} \int_M\!\div(\xi)\varOmega_{(M, g)} &= \int_M\!L_\xi\varOmega_{(M, g)}\\ &= \int_M\!\dd i_\xi\varOmega_{(M, g)}\\ &= \int_{\partial M}\!i_\xi \varOmega_{(M, g)}. \end{align*}\end{split}\]

      • 最後の等式はストークスの定理による。

    2. ベクトル場 \(n\) は定義域を多様体全体に拡張してよい。次のように決めて構わない：

      \[i_n\varOmega_{(M, g)}|\partial M = \varOmega_{(\partial M, g|\partial M)}.\]

    3. 正規直交基底 \(\set{e_i}\ {(i = 1, \dotsc, n)}\) をとる。ただし、境界に沿って局所的に \({n = e_1}\) となるようなものとする。このとき \({\xi = \sum a_i e_i}\) について：

      \[\begin{split}\begin{align*} i_\xi\varOmega_{(M, g)}|\partial M &= a_1 e_2^* \wedge \dotsb \wedge e_n^*\\ &= g(n, \xi)\varOmega_{(\partial M, g|\partial M)}. \end{align*}\end{split}\]

      これを 1. の右辺に適用すればガウス・グリーンの公式を得る。

  • 例 4.4.3: ガウス・グリーンの公式の特殊化

    • 領域 \({B \subset \RR^2}\) に対して次が成り立つ：

      \[\int_B\!\left(\frac{\partial \xi_1}{\partial x_1} + \frac{\partial \xi_2}{\partial x_2}\right)\,\dd x_1\dd x_2 = \int_{\partial B}\!n \cdot \xi\,\dd s.\]

      ただし \(s\) は領域の境界 \({\partial B}\) の向きに沿ったパラメーターとする。

    • 領域 \({B \subset \RR^3}\) に対して次が成り立つ：

      \[\int_B\!\left(\frac{\partial \xi_1}{\partial x_1} + \frac{\partial \xi_2}{\partial x_2} + \frac{\partial \xi_3}{\partial x_3}\right)\,\dd x_1\dd x_2\dd x_3 = \int_{\partial B}\!n \cdot \xi\,\dd S.\]

      ただし \({\dd S}\) は領域の境界 \({\partial B}\) の「面積要素」である。

  • 一般の \(k\) 形式に対しても \(\displaystyle \sum_{i_1 \lt \dotsb \lt i_k} f_{i_1 \dots i_k} \alpha^{(i_1)} \wedge \dotsb \wedge \alpha^{(i_k)}\) と書いたときの \(\displaystyle \sum_{i_1 \lt \dotsb \lt i_k} f_{i_1 \dots i_k}^2\) の値は正規直交基底のとり方に依存しないで定まる。

    • したがって「長さ」も定まると言いたい？

  • 問題 4.4.4: 行列式の計算

    • 行列 \(A\) を \({m \times n}\) サイズ、

    • 行列 \(B\) を \({n \times m}\) サイズで

    あるとする。このとき \(\det(AB)\) はどう書けるかという問題。

    • \({A = (a_{ij}),}\)

    • \({B = (b_{jk}),}\)

    • \(i, k = 1, \dotsc, m,\)

    • \(j = 1, \dotsc, n\)

    とおく。

    • (1) \({m \ge n}\) ならば \({\det(AB) = 0.}\)

      \(AB\) のランクのことを考えれば明らか。

    • (2) \({m \le n}\) ならば：

      \[\begin{split}\det(AB) = \sum_{j_1 \lt \dotsb \lt j_m} \det((a_{ik})_{\substack{i = 1, \dots, m\\k = j_1, \dotsc, j_m}}) \det((b_{ki})_{\substack{i = 1, \dots, m\\k = j_1, \dotsc, j_m}})\end{split}\]

      1. 行列の積を考える：

        \[AB = \left(\sum_{j = 1} a_{ij}b_{jk}\right)_{i, k = 1, \dotsc, m}.\]

      2. 行列式をひたすら考える：

        \[\begin{split}\begin{align*} \det(AB) &= \sum_\sigma \sgn(\sigma) \left(\sum_{j_1 = 1} a_{1 j_1} b_{j_1 \sigma(1)}\right) \dotsm \left(\sum_{j_m = 1} a_{m j_m} b_{j_m \sigma(m)}\right)\\ &= \sum_\sigma \sum_{J \subset \set{1, \dotsc, n}} \sum_{\set{j_1, \dotsc, j_m} = J} \sgn(\sigma) a_{1 j_1} b_{j_1 \sigma(1)} \dotsm a_{m j_m} b_{j_m \sigma(m)}\\ &= \sum_\sigma \sum_{j_1 < \dotsb < j_m} \sum_\tau \sgn(\sigma) a_{1 j_{\tau(1)}} b_{j_{\tau(1)} \sigma(1)} \dotsm a_{m j_{\tau(m)}} b_{j_{\tau(m)} \sigma(m)}. \end{align*}\end{split}\]

      3. シグマを一個取った部分を計算すると：

        \[\begin{split}\begin{align*} \sum_\sigma \sum_\tau \sgn(\sigma) a_{1 j_{\tau(1)}} b_{j_{\tau(1)} \sigma(1)} \dotsm a_{m j_{\tau(m)}} b_{j_{\tau(m)} \sigma(m)} &= \sum_\tau \sgn(\tau) a_{1 j_{\tau(1)}} \dotsm a_{m j_{\tau(m)}} \sum_\sigma \sgn(\sigma)\sgn(\tau) b_{j_{\tau(1)} \sigma(1)} \dotsm b_{j_{\tau(m)} \sigma(m)}\\ &= \sum_\tau \sgn(\tau) a_{1 j_{\tau(1)}} \dotsm a_{m j_{\tau(m)}} \sum_\sigma \sgn(\sigma\tau\inv) b_{j_1 \sigma(\tau\inv(1))} \dotsm b_{j_m \sigma(\tau\inv(m))}. \end{align*}\end{split}\]

      4. 再び \(\displaystyle \sum_{j_1 \lt \dotsb \lt j_m}\) を適用すると所望の結論を得る。

  • 正規直交基底 \(\set{\alpha^{(1)}, \dotsc, \alpha^{(n)}}\) に対して、 \({\alpha^{(i_1)} \wedge \dotsb \wedge \alpha^{(i_k)}}\) が \(k\) 次外積の空間における自然な内積についての正規直交基底になっていることが今のでわかる。自然な内積とは次のものだ：

    \[\sum_{i_1 \lt< \dotsb \lt i_k} f_{i_1 \dots i_k} \alpha^{(i_1)} \wedge \dotsb \wedge \alpha^{(i_k)}, \sum_{i_1 \lt< \dotsb \lt i_k} g_{i_1 \dots i_k} \alpha^{(i_1)} \wedge \dotsb \wedge \alpha^{(i_k)} \longmapsto \sum_{i_1 \lt< \dotsb \lt i_k} f_{i_1 \dots i_k} g_{i_1 \dots i_k}.\]

  • 微分形式同士の内積を定義する。

    • 多様体 \(M\) は向き付けられたコンパクト閉多様体であり、

    • \({\alpha, \beta \in \varOmega^k(M)}\) であり、

    • \({(\alpha, \beta)}_x\) を \({\extp^k T_x^*M}\) の内積である

    とすると、次で定義される：

    \[(\alpha, \beta) = \int_M\!(\alpha, \beta)_x\varOmega_{(M, g)}.\]

  • ホッジのスター作用素

    \(\fn{*}{\extp^k T^*M}\extp^{n - k}T^*M\) を次のように定義する：

    \[\begin{split}\ast(\alpha^{(i_1)} \wedge \dotsb \wedge \alpha^{(i_k)}) = \sgn \begin{pmatrix} 1 & \cdots & \cdots & \cdots & \cdots & n\\ i_1 & \cdots & i_k & j_1 & \cdots & j_{n - k} \end{pmatrix} \alpha^{(j_1)} \wedge \dotsb \wedge \alpha^{(j_{n - k})}\end{split}\]

    • ここで各 \(\alpha^{(\cdot)}\) は正の向きの正規直交基底であり、

    • \({i_1 \lt \dotsb \lt i_k,}\ {j_1 \lt \dotsb \lt j_{n - k}}\) であり、

    • \(\sgn\) ホニャララは \(n\) 個の添字の置換の符号を意味するものとする。

  • 問題 4.4.5: スター作用素の定義は正規直交基底のとり方に依存しない

    1. 正の向きの正規直交基底 \(\set{\alpha^{(\cdot)}}, \set{\beta^{(\cdot)}}\) に対して次の等式を満たす行列 \({A = (a_{ij}) \in SO(n)}\) が存在する：

      \[\beta^{(i)} = \sum_{j = 1}^n a_{ij}\alpha^{(j)}.\]

    2. \({\ast(\beta^{(i_1)} \wedge \dotsb \wedge \beta^{(i_k)})} = {P \alpha^{(l_1)} \wedge \dotsb \wedge \alpha^{(l_{n - k})}}\) の形に書き表す。 \(P\) の部分は本書にあるようにゴチャゴチャしている。

    3. \(\displaystyle {\alpha^{(l)} = \sum_{m = 1}^n a_{ml}\beta^{(m)}}\) を用いて 2. の \({\alpha^{(l_1)} \wedge \dotsb \wedge \alpha^{(l_{n - k})}}\) を \({Q \beta^{(m_1)} \wedge \dotsb \wedge \beta^{(m_{n - k})}}\) の形に書き表す。\(Q\) の部分はやはりゴチャゴチャしている。

    4. 3. を 2. に代入して次のように変形したい：

      \[\begin{split}\sgn\begin{pmatrix} 1 & \cdots & \cdots & \cdots & \cdots & n\\ i_1 & \cdots & i_k & m_1 & \cdots & m_{n - k} \end{pmatrix} \beta^{(m_1)} \wedge \dotsb \wedge \beta^{(m_{n - k})}\end{split}\]

      それには \(P\) と \(Q\) が上記の置換の符号と一致することを、大量のシグマ記号と置換をうまく捌いて示せば十分。

  • ホッジのスター作用素の性質いろいろ

    • \({\ast 1 = \varOmega_{(M, g)}.}\)

    • \(\ast\) は内積を保つ線形同型写像である。

    • \({\ast \circ * = (-1)^{k(n - k)}.}\)

    • \(\ast\) は写像 \(\fn{*}{\varOmega^k(M)}\varOmega^{n - k}(M)\) を引き起こす。 \(\varOmega^k(M)\) の内積を次のように書ける：

      \[\begin{split}\begin{align*} (\alpha, \beta) &= \int_M\!(\alpha, \beta)_x \varOmega_{(M, g)}\\ &= \int_M\!\alpha \wedge *\beta\\ &= \int_M\!*\alpha \wedge \beta. \end{align*}\end{split}\]

      • \({\alpha \in \varOmega^{k - 1}(M),}\ {\beta \in \varOmega^k(M)}\) とする。

        • 写像 \(\fn{\delta}{\varOmega^k(M)}\varOmega^{k - 1}(M)\) を次のように定義する：

          \[\delta = (-1)^{n(k + 1) + 1}(\ast \circ \dd{} \circ \ast).\]

        このとき \({(\dd\alpha, \beta) = (\alpha, \delta\beta)}\) が成り立つ：

        \[\begin{split}\begin{align*} (\dd\alpha, \beta) &= \int_M\!(\dd\alpha) \wedge \ast\beta\\ &= \int_M\!\dd(\alpha \wedge \ast\beta) - (-1)^{k - 1}\alpha \wedge \dd(\ast\beta)\\ &= -\int_M\!(-1)^{k - 1}\alpha \wedge (-1)^{(n - k + 1)(k - 1)}(\ast \circ \ast)\dd(\ast\beta)\\ &= -(-1)^{(n - k)(k - 1)}(-1)^{n(k + 1) + 1} \int_M\!\alpha \wedge \ast\delta\beta\\ &= \int_M\!\alpha \wedge \ast\delta\beta\\ &= (\alpha, \delta\beta). \end{align*}\end{split}\]

        • 式変形の途中でムリヤリ \(\delta\) を出現させるところが急所。

  • \(\delta\) の性質いろいろ

    • \({\delta \circ \delta = 0}\) であることから \({(\varOmega^*(M), \delta)}\) は複体である。

    • \({(\dd\alpha, \beta)} = {(\alpha, \delta\beta)}\) などが成り立つことから、部分空間の直交性 \({\ker\dd \perp \im\delta,}\ {\im\delta \perp \ker\delta}\) がある。

      • 直交するとは、内積がゼロとなることである。

    • \(\varOmega^k(M)\) には互いに直交する部分空間 \({\ker\dd \cap \ker\delta,}\ {\im\dd},\ {\im\delta}\) が存在する。

    • \(\boldsymbol{H}^k = \set{\alpha \in \varOmega^k(M) \sth {(\dd\delta + \delta\dd)}\alpha = 0}\) とおくと、 \(\boldsymbol{H}^k = {\ker\dd{} \cap \ker\delta}\) が成り立つ。

      • \(\alpha \in \boldsymbol{H}^k\) ならば \(0 = {(\dd\delta + \delta\dd)\alpha} = {(\delta\alpha, \delta\alpha) + (\dd\alpha, \dd\alpha)}\) であるので \({\alpha \in \ker\dd{} \cap \ker\delta}\) と言える。

      • \({\alpha \in \ker\dd{} \cap \ker\delta}\) ならば当然 \({\alpha \in \boldsymbol{H}^k}\) である。

    • \({\Laplace = \dd\delta + \delta\dd}\) と書き、ラプラシアン と呼ぶ。

      • \({\boldsymbol{H}^k = \ker\Delta.}\)

      • \({\Laplace\alpha = 0}\) を満たす \(\alpha\) を 調和形式 という。

        \[\Laplace\alpha = 0 \iff (\dd\alpha = 0) \land (\delta\alpha = 0) \iff \alpha \in \ker\dd \cap \ker\delta.\]

  • 定理 4.4.6: ホッジ・ドラーム・小平の定理

    \({\varOmega^k(M) = \boldsymbol{H}^k \oplus \im\dd{} \oplus \im\delta}\) は直交する部分空間への直和分解である。
