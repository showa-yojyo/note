以下のノートにおいて、当分の間、多様体の話題が出てくるときには多様体、座標近傍系、座標、座標変換、座標近傍の像、多様体間の写像に対していつも使っていた記号を用いる。

  • 定義 2.1.7: 多様体上の微分形式 (1)

    \(\alpha\) が多様体上の微分形式であるとは、各 \({V_i = \varphi(U_i)}\) 上の \(C^\infty\) 級 \(p\) 形式であって、次の条件を満たすものをいう：

    \[\alpha^{(j)}|V_{ij} = \varphi_{ij}^* \alpha^{(i)}.\]

  • 例題 2.1.8: \((U_0, \varphi_0)\) が座標近傍系 \(\set{(U_i, \varphi_i)_{i \in I}}\) と両立するとき、多様体上の微分形式 \(\alpha\) は \({V_0 = \varphi_0(U_0)}\) 上の \(\alpha^{(0)}\) で、 \({\alpha^{(0)}|V_{i0} = \varphi_{i0}^* \alpha^{(i)}}\) を満たすものを定める。

    • まず \(V_0\) の開被覆を \(\set{(V_{i0})}_{i \in I}\) とおく。

      • 座標近傍の像のほうの開被覆をとるのは珍しい気がする。

      示したいことは、\({\forall V_{i0} \cap V_{j0},}\ {\varphi_{i0}^*|\alpha^{(i)} = \varphi_{j0}^*|\alpha^{(i)}}`ならば :math:`V_0\) 上のすべての点において \(\alpha^{(0)}\) が well-defined であることだ。

    • 両立しているから \({i, j, k \in \zeroset \cup I,}\quad {\varphi_{ik} = \varphi_{ij} \circ \varphi_{jk}}\) だが、特に \({k = 0}\) として \({\varphi_{i0} = \varphi_{ij} \circ \varphi_{j0}}\) だ。

    • 定義 2.1.7 の関係が \(V_{ij}\) 上成り立っているので、 \({V_{i0} \cap V_{j0}}\) 上次のようになる：

      \[\begin{split}\begin{align*} \varphi_{i0}^* \alpha^{(i)} &= (\varphi_{ij} \circ \varphi_{j0})^* \alpha^{(i)}\\ &= \varphi_{j0}^* \varphi_{ij}^* \alpha^{(i)}\\ &= \varphi_{j0}^* \alpha^{(j)}. \end{align*}\end{split}\]

以上より、\(p\) 形式 \(\alpha\) は任意の座標近傍 \(U\) 上で、ユークリッド空間の開集合 \(\varphi(U)\) 上の \(p\) 形式として表示されていることがわかった。

2.2 余接空間¶

  • 接ベクトルの定義と似せて、微分形式を関数の同値類として定義する。

  • 多様体上の点において微分可能な関数 \(f_1, f_2\) が同値であることを次のように定義する：

    \[f_1 \sim f_2 \iff \dd{(f_1 \circ \varphi\inv)}_{(\varphi(x))} = \dd{(f_2 \circ \varphi\inv)}_{(\varphi(x))}.\]

  • 座標近傍 \({(U, \varphi)},\ {(V, \psi)}\) について \({f_1 \sim f_2} \implies {\dd{(f_1 \circ \psi\inv)}_{(\psi(x))} = \dd{(f_2 \circ \psi\inv)}_{(\psi(x))}}\) である。

    • このことは \({\dd{(\varphi^*f)} = \varphi^*\dd f}\) や \({f_k \circ \psi\inv} = {f_k \circ \varphi\inv \circ \varphi \circ \psi\inv}\) を使って示せる。

  • 定義 2.2.1: 余接空間

    \[T_x^*M = C^\infty(M)/\sim.\]

  • 例題 2.2.2: 余接空間は \(C^\infty(M)\) の実ベクトル空間から定まるベクトル空間である。

    • 写像 \([f] \in C^\infty(M)/\sim \longmapsto \left(\dfrac{\partial f}{\partial x_1}(\varphi(x)), \dotsc, \dfrac{\partial f}{\partial x_n}(\varphi(x))\right) \in \RR^n\) が準同型写像（線形写像）となる：

      \[\frac{\partial(a_1 f_1 + a_2 f_2)}{\partial x_i}(\varphi(x)) = a_1 \frac{\partial f_1}{\partial x_i}(\varphi(x)) + a_2 \frac{\partial f_2}{\partial x_i}(\varphi(x)).\]

    • この写像は同値類の定義により単射である。

    • 全射であることを示すのに 1 の分割の技法を用いる。

      • まず \(\bm a \in \RR^n\) に対して \(U\) 上の関数 \(f_{\bm a} = \sum_{i = 1}^n a_i x_i\) を対応させる。

      • 定理 2.1.4 の \(\nu\) に対して \(\nu f_{\bm a}\) を考える。これは \(U\) 上の関数 \(f_{\bm a}\) を \(M\) 上に拡張された \(C^\infty\) 級関数となっている。

      \[\dd(\nu f_{\bm a}) = \dd\left(\nu \sum_{i = 1}^n a_i x_i\right)_{\varphi(x)} = \sum_{i = 1}^n a_i (\dd x_i)_{\varphi(x)}.\]

  • 点 \({x \in M}\) の局所座標が \({(x_1, \dotsc, x_n)}\) であるとき、 \(T_x^*M\) の基底を \({\dd x_1, \dotsc, \dd x_n}\) と書く。

    • 点を明示する場合もある。

    • 基底の取り方が座標近傍によることに注意。

  • 基底の変換は 1 形式の引き戻しの式に一致する：

    \[\begin{split}\begin{align*} \dd y_i &= \sum_{j = 1}^n \left(\dfrac{\partial y_i}{\partial x_j}\right)_{(\varphi(x))}\,\dd x_j\\ (\varphi \circ \psi\inv)^* (\dd y_i)_{\psi(x)} &= \sum_{j = 1}^n \left(\dfrac{\partial y_i}{\partial x_j}\right)_{(\varphi(x))}\, (\dd x_j)_{\varphi(x)}. \end{align*}\end{split}\]

  • 定義 2.2.3: 多様体上の 1 形式

    • \(x \in M\)

    • \(f_i \in C^\infty(M)\)

    このとき各座標近傍上での一次結合への対応 \({x \longmapsto \displaystyle \sum_{i = 1}^n f_i}\,\dd x_i \in T_x^*M\) を表題のように呼ぶ。

  • 定義 2.2.4: 多様体上の全微分

    同じ条件で \({\displaystyle \dd f = \sum_{i = 1}^n \dfrac{\partial f_i}{\partial x_i}\,\dd x_i}\) のことを表題のように呼ぶ。

  • 例題 2.2.5: 余接空間の準同型写像（線形写像）

    • \(C^\infty\) 級写像 \(\fn{F}{M}N\)

    • \(C^\infty\) 級関数 \(\fn{f}{N}\RR\)

    このとき \({F^*f = f \circ F}\) は準同型（線形写像） \(\fn{F^*}{T_{F(x)}^*N}T_x^*M\) を引き起こす。

    • 証明は 2.2 節 冒頭の式変形のようにすることで示せる。

    • \(\fn{F^*}{C^\infty(N)}C^\infty(M)\) として準同型であるので、余接空間の写像として見ても準同型である。

  • 問題 2.2.6: ステレオグラフ

    多様体上では 1 形式と接ベクトルとをはっきり区別して考える必要があることを示すデモのような問題だ。単位球面の北極と南極に関して投影写像を次のようにとる：

    \[\begin{split}\fnm{\pi_N}{S^2\setminus\set{p_N}}{\RR^2}{(x_1, x_2, x_3)}(v_1, v_2) = \left(\frac{x_1}{1 - x_3}, \frac{x_2}{1 - x_3}\right),\\ \fnm{\pi_S}{S^2\setminus\set{p_S}}{\RR^2}{(x_1, x_2, x_3)}(u_1, u_2) = \left(\frac{x_1}{1 + x_3}, \frac{x_2}{1 + x_3}\right).\end{split}\]

    • (1) 各投影写像の逆写像

      計算方法は球と直線との交点を求めるだけだ。二次方程式を解くことになり、ニ根のうち 1 でないほうを \(x_3\) とすればよい。SymPy で計算した結果を print_latex で出力し、それを整形したものを記す：

      \[\begin{split}\begin{align*} \pi_N\inv(x_1, x_2, x_3) &= \left( \frac{2 v_1}{v_1^2 + v_2^2 + 1}, \frac{2 v_2}{v_1^2 + v_2^2 + 1}, \frac{v_1^2 + v_2^2 - 1}{v_1^2 + v_2^2 + 1}\right),\\ \pi_S\inv(x_1, x_2, x_3) &= \left( \frac{2 u_1}{u_1^2 + u_2^2 + 1}, \frac{2 u_2}{u_1^2 + u_2^2 + 1}, -\frac{u_1^2 + u_2^2 - 1}{u_1^2 + u_2^2 + 1}\right). \end{align*}\end{split}\]

    • (2) 座標近傍系 \(\set{(S^2\setminus\set{p_N}, \pi_N), (S^2\setminus\set{p_S}, \pi_S)}\) の座標変換

      SymPy の出力を整形したものを記す：

      \[\begin{split}\begin{align*} \pi_N \circ \pi_S\inv(u_1, u_2) &= \left( \frac{u_1}{u_1^2 + u_2^2}, \frac{u_2}{u_1^2 + u_2^2}\right)\\ \pi_S \circ \pi_N\inv(v_1, v_2) &= \left( \frac{v_1}{v_1^2 + v_2^2}, \frac{v_2}{v_1^2 + v_2^2}\right) \end{align*}\end{split}\]

    • (3) 下に示す平面上の多項式係数のベクトル場について \({(\pi_N\inv)_*\xi}\) が \(S^2\) 上でも微分可能である条件とは？

      \[\xi = P(v_1, v_2) \frac{\partial}{\partial v_1} + Q(v_1, v_2) \frac{\partial}{\partial v_2}.\]

      大まかな解き方：

      1. \({\pi_S}_*(\pi_N\inv)_*\xi\) を \(P, Q, u_1, u_2, \dfrac{\partial}{\partial u_1}, \dfrac{\partial}{\partial u_2}\) を使って表す。例えば次を使う：

        \[\frac{\partial}{\partial v_j} = \sum_{i = 1, 2} \frac{\partial u_i}{\partial v_j}\frac{\partial}{\partial u_i}\quad(j = 1, 2).\]

      2. \({k = \max\set{\deg P, \deg Q}}\) とし、 \({\pi_S}_*(\pi_N\inv)_*\xi\) の \({-k + 2}\) 次の項を求める。

        • \({k \gt 2}\) とすると \(\dfrac{\partial}{\partial u_1}, \dfrac{\partial}{\partial u_2}\) の係数の有理式が \({u_1 = u_2 = 0}\) でも有効であるには、分子と分母の両方がゼロであることが必要。つまり \(P, Q\) の \(k\) 次の項を \(P_k, Q_k\) とおくと、これらが両方ゼロであることが必要となる。

      3. \({k = 2}\) とすると \(\dfrac{\partial}{\partial u_1}, \dfrac{\partial}{\partial u_2}\) の係数の有理式はそれぞれ \(A(u_1^2 + u_2^2)^2\) と \(B(u_1^2 + u_2^2)^2\) の形をとる。すなわち：

        \[\begin{split}\begin{align*} P_2(u_1, u_2) &= (u_2^2 - u_1^2)A - 2 u_1 u_2 B,\\ Q_2(u_1, u_2) &= -2 u_1 u_2 A - (u_2^2 - u_1^2)B. \end{align*}\end{split}\]

      4. 一次同次の項を \({P_1(u_1, u_2) = a_1 u_1 + a_2 u_2,}\) \({Q_1(u_1, u_2) = b_1 u_1 + b_2 u_2}\) とすると、引き算して \({P_1 = a_1 u_1 - b_1 u_2,}\ {Q_1 = b_1 u_1 + a_1 u_2}\) ならば 0 次のベクトル場として球面上に拡張できることがわかる。

      5. 最後に積分定数のようなものを考慮に入れて、与えられたベクトル場の形とは次のようなものである：

        \[((v_2^2 - v_1^2)A - 2v_1v_2B + a_1v_1 - b_1v_2 + c_1)\frac{\partial}{\partial v_1} + (-2v_1v_2A - (v_1^2 - v_2^2)B + b_1v_1 + a_1v_2 + c_2)\dfrac{\partial}{\partial v_2}.\]

    • (4) 下に示す平面上の多項式係数の 1 形式について \(\pi_N^* \alpha\) が \(S^2\) 上でも微分可能である条件とは？

      \[\alpha = P(v_1, v_2) \dd v_1 + Q(v_1, v_2) \dd v_2.\]

      大まかな解き方：

      1. \(\dd v_1, \dd v_2\) を \(\dd u_1, \dd u_2\) で表す：

        \[\dd v_j = \sum_{i = 1, 2}\frac{\partial v_j}{\partial x_i}\,\dd x_i\quad(j = 1, 2).\]

      2. \({\pi_S\inv}^* \pi_N^* \alpha\) を \(P, Q, u_1, u_2, \dd u_1, \dd u_2\) で表す。

      3. 今度は \({\pi_S\inv}^* \pi_N^* \alpha\) の \({-k - 2}\) 次の項を求めることになる。

        • \(\dd u_1, \dd u_2\) の係数の有理式が複雑。

        • \({k \ge 0}\) とすると分母に \({(u_1^2 + u_2^2)}^{k + 2}\) が現れる。つまり \({P_k = Q_k = 0}\) が必要。

      4. よって 0 以外の多項式では与えられた微分形式は球面上に拡張できない。

2.3 p 次外積の空間¶

  • 定義 2.3.1: \(p\) 次外積の空間 \(\extp^p T_x^*M\)

    余接空間 \(T_x^*M\) の \(p\) 次外積空間とは、\(1 \le i_1 \lt \dotsb \lt i_p \le n\) なる自然数の添字に対応する記号 \({\dd x_{i_1} \wedge \dotsb \wedge \dd x_{i_p}}\) を基底とするベクトル空間のことをいう。

  • 例 2.3.2: 四次元空間の二次外積空間は六次元ベクトル空間である。

  • 余接空間の基底の取り替えにより、外積空間の基底も座標変換される：

    \[\dd x_{i_1} \wedge \dotsb \wedge \dd x_{i_p} = \sum_{j_1, \dotsc, j_p = 1} \frac{\partial x_{i_1}}{\partial y_{j_1}}\dots \frac{\partial x_{i_p}}{\partial y_{j_p}}\, \dd y_{j_1} \wedge \dotsb \wedge y_{j_p}.\]

    • 同じ添字があれば \({\dd y_{j_1} \wedge \dotsb \wedge y_{j_p} = 0.}\)

    • 添字列が互いに順列ならば、値は符号だけが異なる：

      \[\begin{split}\dd{y_{j_1}} \wedge \dotsb \wedge \dd{y_{j_p}} = \sgn \begin{pmatrix}j_1 & \dots & j_p\\k_1 & \dots & k_p\end{pmatrix} \dd{y_{k_1}} \wedge \dotsb \wedge \dd{y_{k_p}}.\end{split}\]

    • cf. 定義 1.6.8

  • 注意 2.3.3: 外積空間の座標変換は外積と両立するように定義されている。

  • 定義 2.3.4: 多様体上の微分形式 (2)

    • 各座標近傍上で、各点における余接空間の外積空間 \(\extp^p T_x^*M\) の元を、\(f_{i_1\dots i_p}\) が \(C^\infty\) 級であるように、 \(f_{i_1\dots i_p}\,\dd x_{i_1} \wedge \dotsb \wedge \dd x_{i_p}\) に対応させるものをこの多様体上の \(C^\infty\) 級微分 \(p\) 形式という。

    • 記号 \(\varOmega^p(M)\) で \(p\) 形式の空間を表す。多様体が一次元以上であればこれは無限次元のベクトル空間である。

    • cf. 定義 2.1.7

  • 例 2.3.5: \(n\) 次元トーラス \(T^n = \RR^n/\ZZ^n\)

    • これは多様体である：

      \(\fn{\pi}{\RR^n}T^n\) を射影とする。また開集合 \({U \in \RR^n}\) において射影が単射であるならば、そのような \(U\) を集めて \(\set{(\pi(U), (\pi|U)\inv)}\) を構成すると、トーラスの座標近傍系となる。

    • トーラス上の微分形式とは：

      1. \(\RR^n\) の座標を使って書かれる。

      2. 二つの座標近傍の共通部分上において一致する。

      3. ただし \(f_{\dots}(\bm x)\) はすべての \({\bm n \in \ZZ^n}\) について \(f_{\dots}{(\bm x + \bm n)} = f_{\dots}{(\bm x)}\) となる。

        特に \(f_{\dots}\) が定数関数であれば、トーラス上の \(p\) 形式であるといえる。

  • 多様体間の \(C^\infty\) 級写像 \(\fn{F}{M}N\) は 例題 2.2.5 にあるように線形写像 \(\fn{F^*}{T_{F(x)}^*N}T_x^*M\) を引き起こすが、同時に線形写像 \(\fn{F^*}{\extp^p T_{F(x)}^*N} \extp^p T_x^*M\) を引き起こす。

    \[\begin{split}\begin{align*} F^*((\dd y_{i_1} \wedge \dotsb \wedge \dd y_{i_p})_{F(x)}) &= F^*(\dd y_{i_1})_{F(x)} \wedge \dotsb \wedge F^*(\dd y_{i_p})_{F(x)}\\ &= \dd (y_{i_1} \circ F)_x \wedge \dotsb \wedge \dd (y_{i_p} \circ F)_x. \end{align*}\end{split}\]

    引き戻しが定義できることがわかる。

  • 命題 2.3.6: 引き戻し \(\fn{F^*}{\varOmega^p(N)}\varOmega^p(M)\)

    • 定義 1.8.3 と同じ命題？

  • 命題 2.3.7: 反変性 \({(F \circ G)^* = G^* \circ F^*}\)

    • \(C^\infty\) 級写像 \(\fn{F}{M}N\) および \(\fn{G}{L}M\) に対して定まる引き戻し \(\fn{F^*}{\varOmega^p(N)}\varOmega^p(M)\) および \(\fn{G^*}{\varOmega^p(M)}\varOmega^p(L)\) について表題の等式が成り立つ。

    • cf. 例題 1.8.9

  • 例 2.3.8: 制限と射影

    • \(\RR^n\) 内の開集合 \(U\) と多様体 \(M^m\) について \({M^m \subset U}\) ならば包含写像 \(\fn{\iota}{M^m}U\) により開集合上の微分形式は多様体上の微分形式に引き戻される。

      \[\alpha \in \varOmega^p(U) \longmapsto \iota^*\alpha \in \varOmega^p(M^m).\]

      • 実は任意の微分形式についてある近傍のある微分形式の制限となっている。これは 第 1 巻問題 5.2.5 を利用して示せるらしい。

    • \({\alpha \in \varOmega^p(T^n)}\) の引き戻し \({\pi^*\alpha \in \varOmega^p(\RR^n)}\) は \(\alpha\) を \(\RR^n\) 上で表示する \(p\) 形式である。

      • 「\(\ZZ^n\) 周期的」という修飾があるが……。

  • 定義 2.3.9 外積

    次の対応は準同型（線形写像）である：

    \[\fnm{\wedge}{\extp^p T_x^*M \times \extp^q T_x^*M}{\extp^{p + q}T_x^*M} {(\dd x_{i_1} \wedge \dotsb \wedge \dd x_{i_p},\ \dd x_{j_1} \wedge \dotsb \wedge \dd x_{j_q})} \dd x_{i_1} \wedge \dotsb \wedge \dd x_{i_p} \wedge \dd x_{j_1} \wedge \dotsb \wedge \dd x_{j_q}.\]

    • cf. 定理 1.7.1

  • 定義 2.4.3: 多様体のドラーム複体

    • 先ほどの図式、正式に言うと多様体の微分形式の コチェイン複体 を多様体 \(M\) のドラーム複体と呼ぶ。

    • 記号は \(\varOmega^*(M)\) とする。

    • 複体というのは性質 \({\dd \circ \dd = 0}\) のことだ。

    • \({\im(\dd) \subset \ker(\dd)}\) の「差」をはかるのにコホモロジー群という群を使う。

  • 定義 2.4.4: ドラーム・コホモロジー 群

    \[\begin{split}\begin{align*} \H^p(M) &= \ker(\fn{\dd}{\varOmega^p(M)}\varOmega^{p + 1}(M))/ \im(\fn{\dd}{\varOmega^{p - 1}(M)}\varOmega^p(M))\\ &= Z^p(M)/B^p(M). \end{align*}\end{split}\]

    • \(\H^p(M)\) は商空間である。

      線形写像の \(\im\) と \(\ker\) の性質によると \(B^p(M)\) と \(Z^p(M)\) はどちらも \(\varOmega^p(M)\) の部分ベクトル空間である。ここで \(B^p(M)\) は \(Z^p(M)\) の部分空間でもあるので、この商空間が成立する。

    • \({Z^p(M) = \ker(\fn{\dd}{\varOmega^p(M)}\varOmega^{p + 1}(M))}\) の元を 閉 p 形式 という。

    • \({B^p(M) = \im(\fn{\dd}{\varOmega^{p - 1}(M)}\varOmega^p(M))}\) の元を 完全 p 形式 という。

    • \({[\alpha] \in \H^p(M)}\) を \({\alpha \in \varOmega^p(M)}\) の コホモロジー 類という。

    • \({\H^*(M) = \bigoplus_{p = 0}^n \H^p(M)}\) と書く。

      • このベクトル空間の直和は何を意味するのかが今はわからない。

  • 例 2.4.5:

    • (1) \(\H^0(M)\) は \(M\) の連結成分で定数となる関数全体のなすベクトル空間である。これは \({\forall f \in Z^0(M)}\) が局所的定数関数であることによる。

    • (2) 星型 \({U \subset \RR^n}\) に対して次が成り立つ：

      \[\begin{split}\H^p(U) \begin{cases} \cong \RR & \text{if } p = 0,\\ = 0 & \text{if } p \gt 0. \end{cases}\end{split}\]

    • \(\varOmega^1(S^1) = Z^1(S^1).\)

    • \(f(t)\,\dd t \in B^1(S^1) \iff \displaystyle \int_0^1\! f(t)\,\dd t = 0.\) 整数周期性による。

    • 次の対応が同型である：

      \[[\alpha] \longmapsto \int_0^1\!\alpha.\]

  • 例 2.4.8: \(\H^*(T^2)\) をフーリエ展開を利用して求める。

    1. \({\H^0(T^2) \cong \RR}\) である。例 2.4.5 (1) による。

    2. \(H^1(T^2)\) を求める。

      • 例 2.3.5 によると \(T^2\) 上の微分形式は \(\RR^2\) 上の周期関数を係数とする微分形式で表される。そこで \({\alpha = g_1}\,{\dd x_1 + g_2}\,{\dd x_2 \in \varOmega^1(T^2)}\) として、各関数をフーリエ級数の形式で表示する：

        \[\begin{split}g_1 = \sum a_{n_1 n_2} \exp(2\pi\sqrt{-1}(n_1 x_1 + n_2 x_2))\\ g_2 = \sum b_{n_1 n_2} \exp(2\pi\sqrt{-1}(n_1 x_1 + n_2 x_2))\end{split}\]

      • ここで \(g_1, g_2\) の性質を確認する。

        • \({g_1 \in C^\infty(T^2)}\) の条件は \(\forall r \gt 0, \sum(n_1^2 + n_2^2)^{(r/2)}\abs{a_{n_1 n_2}} \lt \infty\) である。 \(g_2\) についても同様の条件である。

        • \(g_1 \in \RR\) の条件は \(a_{-n_1 -n_2} = \overline{a_{n_1 n_2}}\) である。\(g_2\) についても同様の条件である。

      • \(\dd\alpha\) を計算することで \({\alpha \in Z^1(T^2)}\) の条件が \({n_1 b_{n_1 n_2} - n_2 a_{n_1 n_2} = 0}\) であることがわかる。

        このことから \({n_1 \ne 0} \implies {b_{n_1 0} = 0}\) と \({n_2 \ne 0} \implies {a_{0 n_2} = 0}\) がわかる。

      • 関数 \({f = \sum c_{n_1 n_2} \exp(2\pi\sqrt{-1}(n_1 x_1 + n_2 x_2))}\) が \(\dd f = \alpha\) を満たす条件を探る。

        \[\dd f = 2\pi\sqrt{-1}\left(\sum n_1 c_{n_1 n_2} \exp(\dots)\,\dd x_1 + \sum n_2 c_{n_1 n_2} \exp(\dots)\,\dd x_2\right).\]

        したがって次が条件である：

        • \({a_{n_1 n_2} = 2\pi\sqrt{-1} n_1 c_{n_1 n_2}}\)

        • \({b_{n_1 n_2} = 2\pi\sqrt{-1} n_1 c_{n_1 n_2}}\)

      • 先ほどの \(\dd\alpha\) の計算結果により、\({\dd f = \alpha}\) には \({b_{00} = a_{00} = 0}\) が必要。このとき \({n_1 \ne 0}, {n_2 \ne 0}\) ならば、次のように係数をおけば \({n_1 b_{n_1 n_2} = n_2 a_{n_1 n_2}}\) だから適切である。

        \[c_{n_1 n_2} = \frac{a_{n_1 n_2}}{2\pi\sqrt{-1}n_1} = \frac{b_{n_1 n_2}}{2\pi\sqrt{-1}n_1}.\]

      • 次の場合分けで \(c_{n_1 n_2}\) が \(c_{00}\) を除いて定まる：

        \[\begin{split}n_1 \ne 0 \implies c_{n_1 0} = \frac{a_{n_1 0}}{2\pi\sqrt{-1}n_1}\\ n_2 \ne 0 \implies c_{0 n_2} = \frac{b_{0 n_2}}{2\pi\sqrt{-1}n_1}.\end{split}\]

      • \({c_{00} = 0}\) とすると \(f\) は \(C^\infty\) 級であり（収束評価を示す必要がある）、\({\dd f = \alpha}\) が成り立つ。

      以上より \({\H^1(T^2) \cong \RR^2}\) であり、\(\alpha\) とフーリエ級数の定数項 \({(a_{00}, b_{00})}\) が対応する。

    3. \(\H^2(T^2)\) を求める。

      \({\beta = h}\,{\dd x_1 \wedge \dd x_2 \in \varOmega^2(T^2)}\) などとおいて、同様の検討を行なう。ここで \({\beta = \dd\alpha}\) かつ

      \[h = \sum e_{n_1 n_2}\exp(2\pi\sqrt{-1}(n_1 x_1 + n_2 x_2))\]

      とおく。

      結果だけ書くと \({\H^2(T^2) \cong \RR}\) であり、\(\beta\) とフーリエ級数の定数項 \(e_{00}\) が対応する。

  • 命題 2.4.10: コチェイン写像

    \(C^\infty\) 写像 \(\fn{F}{M}N\) に対する引き戻し \(\fn{F^*}{\varOmega^p(N)}\varOmega^p(M)\) はコチェイン写像である：

    \[F^*\,\dd = \dd\,F^*.\]

    • 命題 2.3.6 と 定理 1.8.11 を参照。

  • 定理 2.4.11: 上記 \(F\) は準同型 \(\fn{F^*}{\H^p(N)}\H^p(M)\) を引き起こす

    • \({\alpha \in Z^p(N)}\) に対して \({\dd \alpha = 0}\) だから \({0 = F^*}\,{\dd \alpha = \dd F^*\alpha}\) であるので、\({F^* \alpha \in Z^p(M).}\)

    • \({\alpha \in B^p(N)}\) に対して \({\alpha = \dd \beta}\) なる \({\beta \in \varOmega^{p - 1}(N)}\) が存在する。一方、\({F^* \alpha = F^*}\,{\dd \beta = \dd F^*\beta}\) より \({F^* \alpha \in B^p(M).}\)

    以上より、次の対応は準同型であると言える：

    \[\fn{F^*}{\ker(\fn{\dd}{\varOmega^p(N)}\varOmega^{p + 1}(N))} \im(\fn{\dd}{\varOmega^{p - 1}(N)}\varOmega^p(N))\]

    • ベクトル空間としても外積代数としても準同型である。

    • 外積 \(\fn{\wedge}{\varOmega^p(M) \times \varOmega^q(M)}\varOmega^{p + q}(M)\) は \({\H^p(M) \times \H^q(M)}\) 上に外積 \(\fn{\wedge}{\H^p(M) \times \H^q(M)}\H^{p + q}(M)\) を定義する。

    • \({[\alpha] \wedge [\beta]} = {[\alpha \wedge \beta]}\) という演算規則が成り立つ。

    • \(C^\infty\) 写像 \(\fn{F}{M}N\) に対して、\({F^*([\alpha] \wedge [\beta]) = F^*([\alpha]) \wedge F^*([\beta])}\) が成り立つ（外積代数の準同型）。

    \(\fn{I_a^{(U)}}{\varOmega^p({[0, 1]} \times M)}\varOmega^{p - 1}({[0, 1]} \times M)\) に対して次が成り立つ：

    \[\dd I_a(\alpha) + I_a(\dd \alpha) = \alpha - \pi^*(\iota_a^*\alpha).\]

    • これを示すには \({(\id \times (\varphi \circ \psi\inv))}^* I_a^{(U)}\alpha^{(U)} = {I_a^{(U)}\alpha^{(U)}}\) を示し、次に 定義 2.1.7 により無印の \(I_a\) が定義される。この \(I_a\) は 命題 1.9.1 と 注意 1.9.2 により上の式を満たす。

    • 座標近傍 \({(U, \varphi)}\), \({(V, \psi)}\) および \({\alpha \in \varOmega^p([0, 1] \times M)}\) の

      • \({[0, 1]} \times \varphi(U)\) における表示 \(\alpha^{(U)}\) の \(\dd x_0\) を含む成分

      を \({(\id \times (\varphi \circ \psi\inv))}^*\) で引き戻すと、\(\alpha\) の

      • \({[0, 1]} \times \psi(V)\) における表示 \(\alpha^{(V)}\) の \(\dd x_0\) を含む成分

      に \({[0, 1]} \times {\psi(U \cap V)}\) 上一致する。なぜならば \({\alpha^{(V)} = (\id \times {(\varphi \circ \psi\inv)})^*\alpha^{(U)}}\) だったから。

  • 定理 2.4.16: \({[0, 1]} \times M \cong \H^p(M)\)

    • \(\fn{\pi}{[0, 1] \times M}M\), \(\fn{\iota_a}{M}{[0, 1]} \times M\) がドラーム・コホモロジー群に誘導する写像 \(\fn{\pi^*}{\H^p(M)}\H^p({[0, 1]} \times M)\), \(\fn{\iota_a^*}{\H^p({[0, 1]} \times M)}M\) は同型である。

    • さらに

      \[\begin{split}\begin{align*} \iota_a^* \pi^* &= \id_{\H^p(M)}\\ \pi^* \iota_a^* &= \id_{\H^p([0, 1] \times M)} \end{align*}\end{split}\]

      である。したがって \({\iota_0^* = (\pi^*)\inv = \iota_1.}\)

    • 証明

      • \({\pi \circ \iota_a = \id_M}\) および 命題 2.3.7 より \({\iota_a^* \pi^* = \id_M^*,}\ {\id_M^* = \id_{\H^p(M)}.}\)

      • \({(\iota_a \circ \pi)^* = \pi^* \circ \iota_a^*}\) および 命題 2.4.15 より

        • \(p \gt 0\) のときは次が成り立つ \({\dd I_a(\alpha) + I_a(\dd \alpha)} = {\alpha - \pi^*(\iota_a^*\alpha),}\ \alpha\) が存在する：

          \[\dd I_a(\alpha) + I_a(\dd \alpha) = \alpha - \pi^*(\iota_a^*\alpha).\]

          • \(\alpha \in Z^p({[0, 1]} \times M)\) とすると、\({\dd \alpha = 0}\) につき \({\dd I_a(\alpha)} = {\alpha - \pi^*(\iota_a^*\alpha).}\)

          • これをコホモロジー類で考えると \({[\alpha] - [\pi^*(\iota_a^*\alpha)] = 0.}\)

          • ゆえに \({\pi^*\iota_a = \id_{\H^p({[0, 1]} \times M)}.}\)

        • \({p = 0}\) のときは \({\alpha \in Z^0({[0, 1]} \times M)}\) は局所的定数関数であるので \({\pi^*{(\iota_a^*\alpha)}}\) と一致する。

  • 定義 2.4.17: \(C^\infty\) ホモトピック

    二つの \(C^\infty\) 級写像 \(\fn{\varphi_0, \varphi_1}{M}N\) が \(C^\infty\) ホモトピックであるとは、次の性質を満たす \(C^\infty\) 級写像 \(\fn{\varphi}{{[0, 1]} \times M}N\) が存在することをいう：

    \[\begin{split}\varphi_0 = \varphi(0, x),\\ \varphi_1 = \varphi(1, x).\end{split}\]

  • 定理 2.4.18: \(\varphi_0, \varphi_1\) がホモトピックならば \(\varphi_0^*, \varphi_1^*\) もホモトピックである

    • \({\varphi_k = \varphi \circ \iota_k\ (k = 0, 1)}\) と

    • 定理 2.4.16 により

    \(\fn{\iota_0^* = \iota_1^*}{\H^p([0, 1] \times M)}\H^p(M)\) である。したがって \({\varphi_0^* = \iota_0^* \varphi^* = \iota_1^* \varphi^* = \varphi_1^*.}\)

  • 問題 2.4.19: \({\RR^m \times M}\) に対し \({\H^p(\RR^m \times M) \cong \H^p(M)}\)

    • \(\fnm{\pi}{\RR^m \times M}{M}{(\bm x, y)}y\)

    • \(\fnm{\iota}{M}{\RR^m \times M}{y}(0, y)\)

    とおくと、\({\pi \circ \iota = \id_M}\) より \({(\pi \circ \iota)}^* = {\iota^*\pi^*} = \id_{\H^p(M)}.\)

    • \(\fnm{\varphi}{{[0, 1]}\times \RR^m}{\RR^m \times M}{(t, \bm x, y)}(t\bm x, y)\) とおいて、ホモトピー

      • \({\varphi_0 = \iota\circ\pi}\)

      • \({\varphi_1 = \id_{\RR^m \times M}}\)

      を与える。

      • \({(\iota\circ\pi)^*} = {\id_{\RR^m \times M}^*} = {\id_{\H^p(\RR^m \times M)}}.\)

    • \({(\iota\circ\pi)^*} = {\pi^*\circ\iota^*}\) だから \(\pi^*, \iota^*\) は同型写像である。
