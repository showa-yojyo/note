1.6 一般の微分形式¶

冒頭の番号が付いていない図式にある「グラフ」がいい。

  • 一般の次元のユークリッド空間の開集合上に p 形式を定義したいので、 \(\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}\) をすべて定義する。

  • 定義 1.6.1: p 形式

    • 各 \(f_{x_{i_1}} \dots f_{x_{i_p}}\) を連続関数とする。

    • この p を微分形式の次数という。

  • 定義 1.6.2: 外積

    • 同じ添字が一つでもあれば \(\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}} = 0.\)

    • 添字の集合が同じだが配列が異なる場合は、それらの置換の符号を外積の違いとする。これは数式で表現したほうがわかりにくいので、あえてこのように書き残しておく。

      • あとで出てくる 定義 1.6.8 のほうが見やすい。

    • 結合則は普通に使える。

  • 例 1.6.3: 標準的 シンプレクティック形式

    • 2 形式 \(\displaystyle \omega = \sum_{i = 1}^n \dd{x_{2i - 1}} \wedge \dd{x_{2i}}\) の n 個の外積は次の 2n 形式になる：

      \[\overbrace{\omega \wedge \dotsb \wedge \omega}^\text{n} = n\,! \dd{x_1} \wedge \dotsb \wedge \dd{x_{2n}}.\]

  • 例題 1.6.4: 次数付き可換性

    \[\beta \wedge \alpha = (-1)^{\alpha\beta}\alpha \wedge \beta.\]

    • \(pq\) 回の隣接ペアの swap で一方が他方に移るから。

    • これをしっかり意識しないとたいていの計算を間違うことになる。

  • 定義 1.6.5: 外微分

    \[\dd{\left(\sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p}\, \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}\right)} = \sum_{i_1 < \dotsb < i_p} \dd{f_{i_1 \dots i_p}} \wedge \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}.\]

  • 例 1.6.6: 標準的接触形式

    \[\begin{split}\begin{align*} \alpha &= \dd{x_{2n + 1}} + \sum_{i = 1}^n x_{2i - 1}\dd{x_{2i}},\\ \dd{\alpha} &= \sum_{i = 1}^n \dd{x_{2i - 1}} \wedge \dd{x_{2i}},\\ \alpha \wedge \overbrace{\dd{\alpha} \wedge \dotsb \wedge \dd{\alpha}}^\text{n} &= n\,! \dd{x_1} \wedge \dotsb \wedge \dd{x_{2n + 1}}. \end{align*}\end{split}\]

  • 例題 1.6.7

    \[\dd(\alpha \wedge \beta) = \dd\alpha \wedge \beta + (-1)^p\alpha \wedge \beta.\]

    • 例によって符号が次数に依ることに注意。

    • 証明方法は具体的に微分形式を定義して計算すればよい。

  • 定義 1.6.8: 微分形式の積分

    • 開集合 \(U \subset \RR^n\) 上の写像 \(\fn{\kappa}{[a_1, b_1] \times \dotsb \times [a_p, b_p]}U\) に沿う積分を次の式で定義する：

      \[\int_\kappa\!\sum_{i_1 < \dotsb < i_p}f_{i_1 \dots i_p}\,\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}} = \int_{a_1}^{b_1}\dots\int_{a_p}^{b_p}\!\sum_{i_1 < \dotsb < i_p}f_{i_1 \dots i_p}( \kappa(t_1, \dotsc, t_p))\det(D\kappa)\,\dd{t_1}\dots\dd{t_p}.\]

      ただし \(D\kappa\) と書いたのは \(\kappa_{i_1}, \dotsc, \kappa_{i_p}\) を \(t_1, \dotsc, t_p\) でヤコビアンにしたもの。スペースの都合でこう書いた。

    • シグマ記号は別の添字の付け方も採用する。

    • ここで各添字が相異なる場合は次の関係式が成り立つ：

      \[\begin{split}\dd{x_{j_1}} \wedge \dotsb \wedge \dd{x_{j_p}} = \sgn \begin{pmatrix}j_1 & \dots & j_p\\i_1 & \dots & i_p\end{pmatrix} \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}.\end{split}\]

    • 実は次のように書け、以降でこれを利用する：

      \[\int_\kappa\!\alpha = \int_\id\!\kappa\alpha.\]

  • 例題 1.6.9: 直方体版ストークスの定理

    • 開集合 \(U\) に沿う積分が \(p + 1\) 次元直方体から \(U\) への写像 \(\kappa\) に沿う積分の和となる：

      \[\int_\kappa\,\dd{\alpha} = \sum_{q = 1}^{p + 1}(-1)^{q - 1} \left( \int_{\kappa(\dots b_q \dots)}\,\alpha - \int_{\kappa(\dots a_q \dots)}\,\alpha \right).\]

    今のところ証明方法は腕力による。あとで別バージョンが紹介される。

    1. \(\dd\alpha\) を計算する。

    2. 左辺から右辺への等式変形の理由：

      • 最初の等号は 定義 1.6.8 による。

      • 次の等号は行列式を第 q 列で展開して書き直すことで得られる。

      • その次の等号は \(t_q\) で積分するとこうなる。

1.7 ユークリッド空間の開集合上の微分形式の空間¶

以下 \(C^\infty\) 級関数、微分形式のみを考える。

  • 約束として 0 形式とは関数とする。

  • 記号 \(\varOmega^p(U)\) で \(U\) 上の p 形式全体を表す。

    • \(\varOmega^p(U)\) は（集合が空集合でなければ）無限次元ベクトル空間である。

    • \(p < 0 \text{ or } n < p \implies \varOmega^p(U) = \zeroset\) とする。

    • 演算 \(\fn{\dd{}}{\varOmega^p(U)}\varOmega^{p + 1}(U)\) は線形写像である。

  • コチェイン複体 の定義。

    • この微分形式全体の空間の系列であって \(\dd{} \circ \dd{} = 0\) であるものを言う。

    • 開集合 \(U \subset \RR^n\) に対する \(\varOmega^p(U)\) を外微分で系列にしたものがコチェイン複体である。この \(U\) を一般の \(C^\infty\) 級多様体に置き換えた系列はドラーム複体と呼ばれるということを後でやる。

  • 定理 1.7.1: \(\fn{\dd \circ \dd}{\varOmega^p(U)}\varOmega^{p + 2}(U)\) は 0 準同型である

    • 基底に対しては \(\dd{(\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}})} = 0.\)

    • 一般には次のようになってやはりゼロだ：

      \[\begin{split}\begin{align*} \dd{(\dd{(f_{i_1 \dots i_p}\, \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}})})} &= \dd{(\dd{f_{i_1 \dots i_p}}\, \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}})}\\ &= \underbrace{\dd{(\dd{f_{i_1 \dots i_p}})}}_\text{0}\, \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}} + \dd{f_{i_1 \dots i_p}} \wedge \underbrace{\dd{(\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}})}}_\text{0}\\ &= 0. \end{align*}\end{split}\]

  • 定理 1.7.2: ポアンカレの補題

    • 星型 \(U\) 上の p 形式 \(\alpha\) が \(\dd{\alpha} = 0\) ならば \(\dd{\beta} = \alpha\) なる p - 1 形式 \(\beta\) が存在する。

    • cf. 問題 1.2.8

    • 証明は後ほど行なう。

1.8 微分形式の引き戻し¶

図 1.6 の状況をいつでも思い浮かべられるようにしておきたい。当分の間、次の記号を用いる：

  • \(V \subset \RR^m,\ W \subset \RR^n\) を開集合とする。

  • \(\fn{\varphi}{V}W\) を \(C^\infty\) 級写像とする。

  • 座標表示を \(\varphi(\bm x) = \bm y,\ y_i = \varphi_i(x_1, \dotsc, x_m)\) とする。

  • \(\fn{f}{W}\RR\) を \(C^1\) 級写像とする。

  • 定義 1.8.1: 1 形式の引き戻し

    \[\begin{split}\begin{align*} \varphi^*\left(\sum_{i = 1}^n f_i\,\dd{y_i}\right) &= \sum_{i = 1}^n\sum_{j = 1}^m f_i \circ \varphi \frac{\partial \varphi_i}{\partial x_j}\,\dd{x_j}\\ &= \sum_{i = 1}^n(\varphi^* f_i)(\varphi^* \,\dd{y_i})\\ &= \sum_{i = 1}^n \varphi^* f_i\,\dd{\varphi_i}. \end{align*}\end{split}\]

    • 最後の等式は形式的なものだ。

    • 関数の引き戻しを \(\varphi^* f = f \circ \varphi\) で定義する。

  • 命題 1.8.2: \(\dd{(\varphi^* f)} = \varphi^* \dd{f}\)

  • 1 形式 \(\alpha\) に対して次が成り立つ：

    \[\int_\gamma\!\varphi^*\alpha = \int_{\varphi \circ \gamma}\!\alpha.\]

  • 定義 1.8.3: p 形式の引き戻し

    \[\begin{split}\alpha = \sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p}\,\dd{y_{i_1}} \wedge \dotsb \wedge \dd{y_{i_p}} \implies\\ \varphi^* \alpha = \sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p} \circ \varphi\,\dd{\varphi_{i_1}} \wedge \dotsb \wedge \dd{\varphi_{i_p}}.\end{split}\]

    ここで各 \(\dd{\varphi_{i_j}}\) は全微分の意味にとること。

  • 例 1.8.4: いろいろ

    • \(V \subset W\) という状況における包含写像 \(\fn{\iota}{V}W\) と \(\iota^* \alpha = \alpha|V\) の関係（説明のみ）。

    • \(m < n\) とすれば \(\fnm{\iota}{\RR^m}{\RR^n}{(x_1, \dotsc, x_m)}(x_1, \dotsc, x_m, 0, \dotsc, 0)\) である。このとき \(\alpha \in \varOmega^p(\RR^n)\) に対する次の値を \(\alpha|\RR^n\) と書くことがある：

      \[\iota^*\alpha = \sum_{i_1 < \dotsb < i_p \le m} f_{i_1 \dots i_p} \circ \iota\,\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}.\]

    • \(\fnm{\pi}{\RR^n}{\RR^m}{(x_1, \dotsc, x_m, \dotsc, x_n)}(x_1, \dotsc, x_m)\) とする。このとき \(\alpha \in \varOmega^p(\RR^m)\) に対して次の式が成り立つ：

      \[\pi^*\alpha = \sum_{i_1 < \dotsb < i_p \le m} f_{i_1 \dots i_p} \circ \pi\,\dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}.\]

      添字が \(m + 1\) 以上の項はない。

  • 問題 1.8.5: 線形写像の引き戻し？

    • 仮定

      • \(\omega = \dd{x_1} \wedge \dotsb \wedge \dd{x_n}\)

      • 線形写像 \(\fn{L}{\RR^n}\RR^n\) の表現行列を \(A = (a_{ij})\) とする。

    • 結論

      • \(L^*\omega = \det A.\)

      • 式変形の途中で出てくる \(\dd{x_{j_1}} \wedge \dotsb \wedge \dd{x_{j_n}}\) がゼロでない値となるには、この添字の配列が n 次の permutation である必要がある。

      • 与式は次のようになる：

        \[\sum_{\sigma \in \mathfrak S^n} \sgn \sigma( a_{1\sigma(1)}\dots a_{n\sigma(n)}) \dd{x_1} \wedge \dotsb \wedge \dd{x_n}.\]

  • 問題 1.8.6: \(\omega = \dd{x_1} \wedge \dd{x_2} + \dd{x_3} \wedge \dd{x_4}\)

    • \(\fnm{L}{\RR^2}{\RR^4}{(u_1, u_2)}\left(\sum_{j = 1, 2}a_{ij}u_j\right)_{i = 1, \dotsc, 4}\)

    • \(L^*\omega = 0\) の条件は何か。

      • これは素直に計算すればわかる。結局 \(\dd{u_1} \wedge \dd{u_2}\) の係数がゼロになる条件を求めることになる。

  • 問題 1.8.7: 引き戻しと外微分の計算 \(\alpha = \dd{z} + x\dd{y}\)

    Todo

    自力でも計算したが、数式は SymPy で生成したい。

    • \(\fnm{F}{\RR^3}{\RR^3}{(x, y, z)}\left(x, y, z - \frac{xy}{2}\right)\) に対する \(F^*\alpha\) の値？

    • \(F\) のヤコビアンと \(\alpha \wedge \dd{\alpha}\) と \(F^*\alpha \wedge \dd{F^*\alpha}\) の値？

    • \(\varphi_t(x, y, z) = (x\cos t - y \sin t, x \sin t + y \cos t, z)\) に対する \(\varphi_t^* F^* \alpha\) の値？

    • \(\fnm{G}{\RR^3}{\RR^3}{(x, y, z)}(x, y, z - xy)\) および \(\fnm{H}{\RR^3}{\RR^3}{(x, y, z)}(x, y \cos x - z \sin x, y \sin x + z \cos x)\) に対して \(H^* G^* \alpha\) の値？

    • \(\det D(G \circ H)\) および \(H^* G^* \alpha \wedge \dd{H^* G^* \alpha}\) の値？

      • ヤコビアンの行列式は 1 となるが、途中 chain rule に注意。

      • 後半は三角関数の simplify に注意。

  • 微分形式の引き戻しの外積は外積の引き戻しである。

  • 例題 1.8.8: \(\varphi^*(\alpha \wedge \beta) = \varphi^*\alpha \wedge \varphi^* \beta\)

    • \(\alpha = \sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p}\,\dd{y_{i_1}} \wedge \dotsb \wedge \dd{y_{i_p}}\) などと置いて直接計算により示せる。

    • \(\varphi^*(f_{i_1 \dots i_p} g_{j_1 \dots j_q}) = (f \circ \varphi)(g \circ \varphi)\) を利用する。

  • 例題 1.8.9: \(\psi^* \varphi^*\alpha = (\varphi \circ \psi)^*\alpha\)

    • \(\alpha\) を上の例題のようにとり、与式の右辺を展開していく。

      \[(\varphi \circ \psi)^*\alpha = \sum_{j_1 < \dotsb < j_p} f_{j_1 \dots j_p} \circ \varphi \circ \psi\, \dd{(\varphi_{j_1} \circ \psi)} \wedge \dotsb \wedge \dd{(\varphi_{j_p} \circ \psi)}.\]

    • 次に \(\displaystyle \varphi_{j_i} \circ \psi = \sum_{k = 1}^m \frac{\partial \varphi_{j_i}}{\partial x_k}\,\dd{\psi_k}\) か。

    • この次の式変形が思いつかない。 \(\dd{\psi_?}\) を \(\dd{x_?}\) と書き換えたい。全体を \(\psi^*(\quad)\) で囲むような。

  • 定理 1.8.10: 積分の変形？

    • \(C^\infty\) 級写像 \(\fn{\varphi}{V}W\) および

    • \(C^\infty\) 級写像 \(\fn{\kappa}{[a_1, b_1] \times \dotsb \times [a_p, b_p]}V\) と

    • \(W \subset \RR^n\) 上の p 形式 \(\alpha\)

    に対して次が成り立つ：

    \[\int_\kappa\!\varphi^*\alpha = \int_{\varphi \circ \kappa}\!\alpha.\]

    • 証明中の式変形では p. 31 の文章中の等式（下に書く）と 例題 1.8.9 の結果を利用する。

      \[\int_\kappa\!\varphi^*\alpha = \int_{\id}\!\kappa^*\varphi^*\alpha.\]

    • \(\id\) が出てくるのがミソか。

  • 定理 1.8.11: 外微分と引き戻しは可換 \(\dd{\varphi^*\alpha} = \varphi^*\dd{\alpha}\)

    • 外微分と引き戻しはどちらも直方体からの写像に沿う積分と整合するように定義されたものだった。

    • 証明はいつものように \(\alpha\) をおいて、順次展開する：

      \[\begin{split}\begin{align*} \dd{(\varphi^*\alpha)} &= \dd{\left(\varphi^* \sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p}\,\dd{y_{i_1}} \wedge \dotsb \wedge \dd{y_{i_p}}\right)}\\ &= \dd{\left(\sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p} \circ \varphi\,\dd{\varphi_{i_1}} \wedge \dotsb \wedge \dd{\varphi_{i_p}}\right)}\\ &= \sum_{i_1 < \dotsb < i_p} \dd{(f_{i_1 \dots i_p} \circ \varphi)} \wedge \dd{\varphi_{i_1}} \wedge \dotsb \wedge \dd{\varphi_{i_p}}\\ &= \varphi^*\left(\sum_{i_1 < \dotsb < i_p} \dd{f_{i_1 \dots i_p}} \wedge \dd{y_{i_1}} \wedge \dotsb \wedge \dd{y_{i_p}}\right)\\ &= \varphi^*\dd{\alpha}. \end{align*}\end{split}\]

  • 例題 1.6.9 の別解を紹介している。

    • \(\alpha\) が p 形式であれば \(\kappa^*\alpha\) もそうなので、トリッキーだと思うが次の式のようにおく：

      \[\kappa^*\alpha = \sum_{q = 1}^{p + 1} f_q\,\dd{t_1} \wedge \dotsb \wedge \dd{t_{q - 1}} \wedge \dd{t_{q + 1}} \wedge \dotsb \wedge \dd{t_{p + 1}}.\]

    • 次に \(\kappa^*(\dd{\alpha})\) を計算すると、次に示す微分形式の和である：

      \[\sum_{q = 1}^{p + 1} (-1)^{q - 1} \frac{\partial f_q}{\partial t_q} \dd{t_1} \wedge \dotsb \wedge \dd{t_{p + 1}}.\]

    • あとは \(\dd\alpha\) を \(\kappa\) に沿って積分していくと 例題 1.6.9 の右辺が得られる。

      • この過程で 定義 1.8.11, 部分積分、 定理 1.8.10 を利用するようだ。途中で行列式が一切出て来ないのが特徴か？

1.9 ポアンカレの補題の証明¶

  • 命題 1.9.1 \(\dd{I(\alpha)} + I(\dd{\alpha}) = \alpha - \pi^*(\iota_0^*\alpha)\)

    • \(U \subset \RR^n\) を開集合、

    • \(\alpha\) を直積 \([0, 1] \times U\) 上の p 形式、

    • \(\fnm{\iota_0}{V}{[0, 1] \times U}{\bm x}(0, \bm x)\)

    • \(\fnm{\pi}{[0, 1] \times U}{U}{(x_0, \bm x)}x_0\bm x\)

    このとき \(I(\alpha)\) を次のようにおくと表題の等式が成り立つ（ただし \(\alpha\) をいつもようにおいたものとする）：

    \[I(\alpha) = \sum_{0 < i_2 < \dotsb < i_p}\left( \int_0^{x_0}\! f_{0 i_2 \dots i_p}\,\dd{x_0} \right) \dd{x_{i_2}} \wedge \dotsb \wedge \dd{x_{i_p}}.\]

    • これは \([0, 1] \times U\) 上の p - 1 形式である。

    • \(\dd{I(\alpha)} + I(\dd{\alpha}) = \alpha - \alpha_0\) の形になる。ここで \(\alpha_0\) は次のようなものだ：

      \[\alpha_0 = \sum_{0 < i_1 < \dotsb < i_p} f_{i_1\dots i_p} (0, x_1, \dotsc, x_n)\, \dd{x_{i_1}} \wedge \dotsb \wedge \dd{x_{i_p}}.\]

    • 例 1.8.4 より \(\alpha_0 = \pi^*(\iota_0^*\alpha).\)

    証明手順は次のとおり：

    1. \([0, 1] \times U\) 上の p 形式 \(\beta\) を次のようにおく。意味はこの微分形式が \(\varphi(x_0, \bm x) = x_0(\bm x - \bm y) + \bm y)\) による \(\alpha\) の引き戻しとなることにある：

      \[\beta = \sum_{i_1 < \dotsb < i_p} f_{i_1 \dots i_p}(x_0(\bm x - \bm y) + \bm y) (x_0\,\dd{x_{i_1}} + (x_{i_1} - y_{i_1})\,\dd{x_0}) \wedge \dotsb (x_0\,\dd{x_{i_p}} + (x_{i_p} - y_{i_p})\,\dd{x_0}).\]

    2. \(\varphi\) は \(x_0 = 0\) において定数写像 \(\bm y\) であり、 \(x_0 = 1\) において恒等写像 \(\id_U\) である。

      本書図 1.7 のちくわぶが直積 \([0, 1] \times U\) を表している。

    3. \(\dd{\alpha} = 0 \implies \dd{\beta} = \dd{(\varphi^* \alpha)} = \varphi^*(\dd{\alpha}) = 0.\)

      この等式変形で 定理 1.8.11 (p. 31) を利用した。

    4. ゆえに \(\dd{I(\beta)} = \beta - \beta_0\) である。前ページ 命題 1.9.1 による。

    5. \(p > 0\) としたのだから \(\beta_0 = \pi^*(\iota_0^*\beta) = \pi^*0 = 0\) により \(\dd{I(\beta)} = \beta.\)

    6. \(x_0 = 1\) とすると写像 \(\fn{\iota_1}{U}[0, 1] \times U\) による引き戻しによって \(\alpha = \iota_1^*\beta = \iota_1^* \dd{(I(\beta))} = \dd{(\iota_1^* I(\beta))}.\)

  • 注意 1.9.2

    特に \(a \in [0, 1]\) に対して \(I_a(\alpha)\) を上に倣って定義し、 \(\iota_a(\bm x) = (a, \bm x)\) と定義すると下の式が成り立つ：

    \[\dd I_a(\alpha) + I_a(\dd \alpha) = \alpha - \pi^*(\iota_a^*\alpha).\]
