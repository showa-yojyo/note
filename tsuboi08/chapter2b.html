2.5 関手という見方¶

  • 本書では 関手 の正式な定義は与えていない。

  • 下に挙げるような、

    1. 対象 と

    2. 合成写像と恒等写像があるような 射

    の組を抽象化する理論があるらしい。以下その例：

    • \(C^\infty\) 級多様体全部とそれらの間の \(C^\infty\) 級写像の全体

    • ベクトル空間全部と線形写像全部

    • コチェイン複体全部とコチェイン写像全部

  • 多様体に微分形式を与える対応は（多様体、写像）から（コチェイン複体、コチェイン写像）への 反変関手 であるといえる。

    • \(\fn{F}{M}N\) から \(\fn{F^*}{\varOmega^*(N)}\varOmega^*(M)\) への対応をいう。

    • \({(F_2 \circ F_1)^* = F_1^* \circ F_2^*}\) という性質がある。

  • コホモロジー群をとる対応は（コチェイン複体、コチェイン写像）から（ベクトル空間、線型写像）への 共変関手 である。

    • \(\fn{F^*}{\varOmega^*(N)}\varOmega^*(M)\) から \(\fn{F^*}{\H^p(N)}{\H^p(M)}\) への対応をいう。

  • この二つの結合（反変関手となる）のことが 定理 2.4.11 である。

  • やはり同定理より \(M\) と \(N\) が微分同相であれば \({\H^p(M) \cong \H^p(N)}\) である。

    • 微分同相でなくとも、

      • \(\fn{f}{M}N,\ \fn{g}{N}M\) が存在して、

      • 合成写像 \({f \circ g}\) と恒等写像 \(\id_N\) が \(C^\infty\) ホモトピックであり、

      • 合成写像 \({g \circ f}\) と恒等写像 \(\id_M\) が \(C^\infty\) ホモトピックである

      ならば \({\H^p(M) \cong \H^p(N)}\) といえる。

  • 注意 2.5.1: 上記 \(f, g\) は実は連続写像にまで緩められる。\(M\) と \(N\) がホモトピー同値であれば、ドラーム・コホモロジー群は同型となる。

2.6 マイヤー・ビエトリス完全系列¶

  • ドラーム・コホモロジー群の計算ができれば、二つの多様体が異なることを示せる。

  • 命題 2.6.1

    • \(M\) をコンパクト多様体の開集合、

    • \({M_1 \subset M,}\ {M_2 \subset M}\) を \({M_1 \cup M_2 = M}\) なる開集合、

    • \({M_{12} = M_1 \cap M_2}\) とし、

    • \(\fn{i_1}{M_{12}}M_1\), \(\fn{i_2}{M_{12}}M_2\), \(\fn{j_1}{M_1}M\), \(\fn{j_2}{M_2}M\) を包含写像とする。

    このとき、次の写像の列は完全系列である：

    \[\require{amscd} \begin{CD} 0 @>>> \varOmega^p(M) @>{(j_1^*,\ j_2^*)}>> \varOmega^p(M_1) \oplus \varOmega^p(M_2) @>{i_1^* - i_2^*}>> \varOmega^p(M_{12}) @>>> 0 \end{CD}\]

    1. \({(j_1^*, j_2^*)}\) が単射であることを示す：

      • \(j_1, j_2\) はそれぞれ \(M_1, M_2\) 上で単射であるので、その引き戻しも単射である。

      • 単射の直積もまた単射である。

      ゆえに \({(j_1^*, j_2^*)}\) は単射である。

    2. \({\im(j_1^*, j_2^*) = \ker(i_1^* - i_2^*)}\) を示す：

      \({\alpha \in \varOmega^p(M)}\) をとる。このとき次のようになる：

      \[\begin{split}\begin{align*} (i_1^* - i_2^*)\circ(j_1^*, j_2^*)\alpha &= (i_1^* - i_2^*)(j_1^*\alpha \oplus j_2^*\alpha)\\ &= i_1^*j_1^*\alpha - i_2^*j_2^*\alpha\\ &= \alpha \circ j_1 \circ i_1 - \alpha \circ j_2 \circ i_2\\ &= \alpha - \alpha = 0. \end{align*}\end{split}\]

    3. \({i_1^* - i_2^*}\) が全射であることを示す：

      これの証明が複雑だ。まず次のように 1 の分割を構成する：

      • \({V_1 \subset \closure{V_1} \subset M_1}\) であり \(V_1\) は開集合であり \(\closure{V_1}\) はコンパクトであるようにとる。

      • \(V_2\) も同様に定義する。

      • \(C^\infty\) 級関数 \(\fn{\lambda_1}{M}\RR\) を次のようにとる：

        • \({0 \le \lambda_1 \le 1}\)

        • \({\lambda_1|(M \setminus V_2) = 1}\)

        • \({\supp \lambda_1 \subset V_1}\)

      • \(C^\infty\) 級関数 \(\lambda_2\) を \({1 - \lambda_1}\) と定義することで \(\set{\lambda_1, \lambda_2}\) は \(\set{M_1, M_2}\) に従属する 1 の分割となる。

        • \({M \setminus M_2}\) の近傍上 \({\lambda_2 = 0}\) である。なぜならば \({\supp \lambda_2 \subset M_2}\) がコンパクト部分集合として成り立つから。

      • すべての \({\alpha \in \varOmega^p(M_{12})}\) に対して \(\lambda_2\alpha\) を \({M \setminus M_2}\) 上ゼロと定義することで \({M_{12} \cup (M \setminus M_2)} = M_1\) 上の \(p\) 形式であるように拡張できる。

        ゆえに \({\lambda_2\alpha \in \varOmega^p(M_1).}\)

      • 同様にして \({-\lambda_1\alpha \in \varOmega^p(M_2).}\)

      \[\begin{split}\begin{align*} (i_1^* - i_2^*)(\lambda_2\alpha - \lambda_1\alpha) &= \lambda_2\alpha + \lambda_1\alpha\\ &= \alpha. \end{align*}\end{split}\]

  • 命題 2.4.10: コチェイン写像 により 命題 2.6.1 の完全系列を縦に \(\dd\) でつないだ写像のグラフは可換となる。

    \[\begin{split}\begin{CD} @. @A{\dd}AA @A{\dd}AA @A{\dd}AA\\ 0 @>>> \varOmega^{p + 1}(M) @>{(j_1^*,\ j_2^*)}>> \varOmega^{p + 1}(M_1) \oplus \varOmega^{p + 1}(M_2) @>{i_1^* - i_2^*}>> \varOmega^{p + 1}(M_{12}) @>>> 0\\ @. @A{\dd}AA @A{\dd}AA @A{\dd}AA\\ 0 @>>> \varOmega^p(M) @>{(j_1^*,\ j_2^*)}>> \varOmega^p(M_1) \oplus \varOmega^p(M_2) @>{i_1^* - i_2^*}>> \varOmega^p(M_{12}) @>>> 0\\ @. @A{\dd}AA @A{\dd}AA @A{\dd}AA\\ 0 @>>> \varOmega^{p - 1}(M) @>{(j_1^*,\ j_2^*)}>> \varOmega^{p - 1}(M_1) \oplus \varOmega^{p - 1}(M_2) @>{i_1^* - i_2^*}>> \varOmega^{p - 1}(M_{12}) @>>> 0\\ @. @A{\dd}AA @A{\dd}AA @A{\dd}AA \end{CD}\end{split}\]

  • \({\alpha \in \varOmega^p(M_{12})}, {\alpha_1 \in \varOmega^p(M_1)}, {\alpha_2 \in \varOmega^p(M_2)}\), \({i_1^*a_1 - i_2^*a_2 = \alpha.}\)

  • \({\dd \alpha_1 \in \varOmega^{p + 1}(M_1)}, {\dd \alpha_2 \in \varOmega^{p + 1}(M_2).}\) これらは \(M_{12}\) 上一致し、どちらもある \({\beta \in \varOmega^{p + 1}(M_{12})}\) を定める。

  • 命題 2.6.2

    1. \({\beta \in \Z^{p + 1}(M)}\) について \([\beta]\) は \(\alpha\) に対する \(\alpha_1, \alpha_2\) の取り方によらない。

      • \(\alpha = {i_1^*\alpha - i_2^*\alpha} = {i_1^*\alpha' - i_2^*\alpha'}\) とおく。すなわち \({i_1^*(\alpha - \alpha')} - {i_2^*(\alpha_2 = \alpha_2')} = 0\) である。

      • 命題 2.6.1 により、\(\exists \gamma \in \varOmega^p(M)\quad\text{s.t.}\quad (j_1^*, j_2^*)\gamma = (\alpha_1 - \alpha_1', \alpha_2 - \alpha_2').\)

      • このとき閉形式 \({\beta' \in \varOmega^{p + 1}(M)}\) が、\({\dd \alpha_1'} \in {\varOmega^{p + 1}(M_1)}, {\dd \alpha_2'} \in {\varOmega^{p + 1}(M_2)}\) が \(M_{12}\) 上で一致することで定まる。

      • したがって \({\beta' - \beta} = {\dd \gamma}\) となる。

      • ゆえに \([\beta] = {[\beta'] \in \H^{p + 1}(M).}\)

    2. \({\alpha \in B^p(M)} \implies {\beta \in B^{p + 1}(M).}\)

      • \({\alpha = \dd \eta}\) とする。

      • \({\exists \eta_1 \in \varOmega^{p - 1}(M_1)}, {\exists \eta_2 \in \varOmega^{p - 1}(M_2)} \quad\text{s.t.}\quad {i_1^*\eta_1 - i_2^*\eta_2} = \eta.\)

      • \({\alpha_1 = \dd \eta_1}, {\alpha_2 = \dd \eta_2}\) ととれる。

      • \({\dd \alpha_1 = 0,} {\dd \alpha_2 = 0.}\)

      • ゆえに \({\beta = 0}\) ととれる。

      • 先ほどの結果を用いると \(\alpha_1, \alpha_2\) の取り方によらず \({\beta \in B^{p + 1}(M).}\)

  • 定義 2.6.3: 連結準同型 \(\fn{\Delta^*}{\H^p(M_{12})}\H^{p + 1}(M)\)

    • 命題 2.6.2 の定め方に従う。

  • 定理 2.6.4: マイヤー・ビエトリス完全系列

    \[\begin{split}\begin{CD} @. @. \cdots @>{i_1^* - i_2^*}>> \H^{p - 1}(M_{12})\\ @.\\ @>{\Delta^*}>> \H^p(M) @>{(j_1^*,\ j_2^*)}>> \H^p(M_1) \oplus \H^p(M_2) @>{i_1^* - i_2^*}>> \H^p(M_{12})\\ @.\\ @>{\Delta^*}>> \H^{p + 1}(M) @>{(j_1^*,\ j_2^*)}>> \cdots \end{CD}\end{split}\]

    • コチェイン複体の完全系列が与えられると、コホモロジー群の完全系列が得られる。

    • 本書 pp. 63-65 が証明であるが、明らかに手間がかかる。次のことを全て（任意の順序で）示せばよいようだ：

      1. \({\Delta^*(i_1^* - i_2^*)} = 0\)

      2. \({(j_1^*,\ j_2^*)\Delta^*} = 0\)

      3. \({(i_1^* - i_2^*)(j_1^*,\ j_2^*)} = 0\)

      4. \({\ker\Delta^* \subset \im(i_1^* - i_2^*)}\)

      5. \({\ker(j_1^*,\ j_2^*) \subset \im\Delta^*}\)

      6. \({\ker(i_1^* - i_2^*) \subset \im(j_1^*,\ j_2^*)}\)

    • 核にあるのが閉形式、像にあるのが完全形式。

    • 全般的に閉形式の性質と p. 62 の図を駆使する。

2.7 球面のドラーム・コホモロジー¶

以下で使う記号：

  • \({S^1 = \RR/\ZZ}\) を円とする。

  • \(\fn{\pi}{\RR}\RR/\ZZ\) をその射影とする。

  • \({M_1 = \pi((0, 1)),}\ {M_2 = \pi((-1/2, 1/2))}\) とおくと \(S^1\) の開被覆となる。

    • 前者は円周を 0 度から 360 度までを、後者は -180 度から 180 度までを表現している。

  • \(M_{12} = {M_1 \cap M_2}\) とおく。

このマイヤー・ビエトリス完全系列を書くと次のようになる：

\[\begin{split}\begin{CD} 0 @>>> \H^0(S^1) @>>> \H^0(M_1) \oplus \H^0(M_2) @>>> \H^0(M_{12})\\ @.\\ @>>> \H^1(S^1) @>>> 0 \end{CD}\end{split}\]

上の完全系列は次のものと同型である：

\[\begin{split}\begin{CD} 0 @>>> \RR @>>> \RR \oplus \RR @>>> \RR \oplus \RR\\ @.\\ @>>> \H^1(S^1) @>>> 0 \end{CD}\end{split}\]

  • この例に限らず、\(\H^0(\cdot)\) は「連結成分上の定数関数」と同一視する。

  • 意味のあるコホモロジー群の最初と最後を見ると \({\H^1(S^1) \cong \RR}\) と言える。

  • 例題 2.7.1: 円の連結準同型

    円周上の \(C^\infty\) 関数 \(\fn{\nu_1}{[0, 1/2]}{[0, 1]}\), \(\fn{\nu_2}{[1/2, 1]}{[0, 1]}\) が次のように定義されているときの \(\fn{\Delta^*}{\H^0(M_{12})}{H^1(S^1)}\) の記述はどのようなものか：

    \[\begin{split}\begin{align*} \nu_1(t) &= \begin{cases} 0 & \quad\text{if } t \in \left[0, \dfrac{1}{6}\right],\\ \text{(unknown)} & \quad\text{if } t \in \left[\dfrac{1}{6}, \dfrac{1}{3}\right],\\ 1 & \quad\text{if } t \in \left[\dfrac{1}{3}, \dfrac{1}{2}\right],\\ \end{cases} \\ \nu_2(t) &= \nu_1\left(t - \frac{1}{2}\right). \end{align*}\end{split}\]

    • まず \(M_1, M_2\) に従属する 1 の分割 \(\lambda_1, \lambda_2\) を適宜構成する：

      \[\begin{split}\begin{align*} \lambda_1(t) &= \begin{cases} \nu_1(t) & \quad\text{if } t \in \left[0, \dfrac{1}{2}\right],\\ 1 - \nu_2(t) & \quad\text{if } t \in \left[\dfrac{1}{2}, 1\right], \end{cases} \\ \lambda_2(t) &= 1 - \lambda_1(t). \end{align*}\end{split}\]

      これが 1 の分割になっていることは、本書の図を見れば納得できる。

    • 次に \({a, b \in \RR}\) を何かとって、関数 \(\fn{M_{12}}\RR\) を次のように定義する：

      \[\begin{split}f(x) = \begin{cases} a & \quad\text{if } t \in \pi\left(\!\left(0, \dfrac{1}{2}\right)\!\right),\\ b & \quad\text{if } t \in \pi\left(\!\left(\dfrac{1}{2}, 1\right)\!\right). \end{cases}\end{split}\]

    • 次のように \(M_1, M_2\) 上の \(C^\infty\) 級関数を構成すると、 \({i_1^* f_1 - i_2^* f_2} = f\) をみたす（暗算で確認できる）：

      \[\begin{split}\begin{align*} f_1(x) &= \lambda_2(t) f(x) = \begin{cases} a(1 - \nu_1(t)) & \quad\text{if } t \in \left[0, \dfrac{1}{2}\right],\\ b \nu_2(t) & \quad\text{if } t \in \left[\dfrac{1}{2}, 1\right], \end{cases} \\ f_2(x) &= -\lambda_1(t) f(x) = \begin{cases} -a\nu_1(t) & \quad\text{if } t \in \left[0, \dfrac{1}{2}\right],\\ -b(1 - \nu_2(t)) & \quad\text{if } t \in \left[\dfrac{1}{2}, 1\right]. \end{cases} \end{align*}\end{split}\]

      \(f_1\) が \(M_1\) 上滑らかであることは \({t = 1/2}\) における微分可能性を確認すればよい。同様に \(f_2\) が \(M_2\) 上滑らかであることは \({t = 0}\) を確認すればよい。

    • \(M_{12}\) 上において \({\dd f_1} = {\dd f_2} = {-a\dd \nu_1} + {b\dd v_2}\) となる。この値を \(\alpha\) とすると次がわかる：

      \[\alpha = -a\dd \nu_1 + b\dd v_2 = \left(-a\diff{\nu_1}{t} + b\diff{\nu_2}{t}\right)\dd t \in \varOmega^1(S^1).\]

    • したがって \({\Delta^*(a, b)} = {[\alpha]}\) である。

  • 注意 2.7.2: 補足

    • \({a = b}\) ならば \(f = {(i_1^* - i_2^*)(a, 0)}\), \({\Delta^*(a, a)} = 0\) (\(\alpha = {\dd(a\lambda_2)}\)).

    • \({a \ne b}\) ならば \({\Delta^*(a, b)}\) が \(\H^1(S^1)\) の基底となる。\({\displaystyle \int_0^1\!\left(-a\diff{\nu_1}{t} + b\diff{\nu_2}{t}\right)\dd t = b - a}\) となって \(\nu_1, \nu_2\) が消去する。

      • \({[\alpha] = [(b - a)\dd t]}\)

    • 2.4.6 も参照。

  • 命題 2.7.3: 一般の多様体次元の \({\H^*(S^k)}\)

    \[\begin{split}\H^p(S^k) \cong \begin{cases} \RR & \quad\text{if } p = 0, k\\ 0 & \quad\text{if } 0 \lt p \lt k \end{cases}\end{split}\]

    • まず \({H^0(S^k) \cong \RR}\) であることは言える。 0 次元のドラームコホモロジー群が球面上の定数関数全体と同一視できるからだ。

    • それ以外の場合については多様体次元 \(k\) についての帰納法で示す。そのために \({M_1 = S^k \setminus \set{p_S}}\), \({M_2 = S^k \setminus \set{p_N}}`とおく。ここで :math:`p_S, p_N\) はそれぞれ南極、北極に相当する点とする。

      • \(M_{12} = {M_1 \cap M_2}\) は \({(-1, 1) \times S^{k - 1}}\) と微分同相だ。

    • 問題 2.4.19 によると \({\H^p(M_{12}) \cong H^p(S^{k - 1})}\) だ。

    • マイヤー・ビエトリス完全系列：

      \[\begin{CD} \cdots @>>> \H^{k - 1}(M_1) \oplus \H^{k - 1}(M_2) @>>> \H^{k - 1}(M_{12}) @>>> \H^k(S^k) @>>> 0 \end{CD}\]

      においては、帰納法の仮定：

      \[\begin{split}\H^p(S^{k - 1}) \cong \begin{cases} \RR & \quad\text{if } p = 0, k - 1\\ 0 & \quad\text{if } 0 \lt p \lt k - 1 \end{cases}\end{split}\]

      より、次の系列と同型と言える：

      \[\begin{CD} \cdots @>>> 0 \oplus 0 @>>> \RR @>>> \H^k(S^k) @>>> 0 \end{CD}\]

      したがって主張が成り立つ。

    • 証明は終わったが次のようなコメントがある。

      • \(\lambda_1, \lambda_2\) を \(M_1, M_2\) に従属する 1 の分割とし、

      • \({[\omega^{k - 1}]}\) を \({\H^{k - 1}(S^{k - 1})}\) の基底である

      とすると、次式が成り立つように \({\H^k(S^k)}\) の基底 \({[\omega^k]}\) をとれる：

      \[[\omega^k] = \Delta^*[\omega^{k - 1}] = [\dd(\lambda_2 \pi^* \omega^{k - 1})].\]

  • 問題 2.7.4: 問題 2.2.6 の状況で

    1. \(\RR^3\) 上の次の微分形式に対する \({(\pi_S\inv)}^*{(\omega|S^2)}\) とは：

      \[\omega = x_1 \dd x_2 \wedge \dd x_3 -x_2 \dd x_1 \wedge \dd x_3 +x_3 \dd x_1 \wedge \dd x_2\]

      • あらゆる計算が面倒なので SymPy の力を借りる：

        u1, u2 = symbols('u1:3')
x1 = 2*u1/(u1**2 + u2**2 + 1)
x2 = 2*u2/(u1**2 + u2**2 + 1)
x3 = (-u1**2 - u2**2 + 1)/(u1**2 + u2**2 + 1)
dx1du1, dx2du1, dx3du1 = x1.diff(u1), x2.diff(u1), x3.diff(u1)
dx1du2, dx2du2, dx3du2 = x1.diff(u2), x2.diff(u2), x3.diff(u2)

omega213 = -x2 * (dx1du1 * dx3du2 - dx1du2 * dx3du1)

print_latex(omega)

        SymPy の計算結果はこうなる：

        \[\frac{4 \dd u_1 \wedge \dd u_2}{(1 + u_1^2 + u_2^2)^2}.\]

    2. 次の微分形式に対して \(\alpha \in {Z^1(\RR^3\minuszero)}\) を示せ。 \({(\pi_S\inv)}^*{(\alpha|S^2\setminus\set{p_N, p_S})} \in {\varOmega^1(\RR^2\minuszero)}\) を展開しろ。

      \[\alpha = \dfrac{x_1 \dd x_2 - x_2 \dd x_1}{x_1^2 + x_2^2}.\]

      • 前半は直接計算により示せる。外微分を計算すればゼロになる（このくらいの簡単な式であれば暗算でわかってもよい）。後半は SymPy で計算してみる：

        d = Differential

dx2 = dx2du1 * d(u1) + dx2du2 * d(u2)

print_latex(alpha)

        次の結果を得る：

        \[\alpha = \frac{u_1 \dd u_2 - u_2 \dd u_1}{u_1^2 + u_2^2}.\]

    3. \(\fnm{\gamma}{[0, 1]}{\RR^3}{t}{(\cos 2\pi t, \sin 2\pi t, 0)}\) に沿った \(\alpha\) の線積分。

      解答で省略されている部分を記す：

      \[\begin{split}\begin{align*} \int_\gamma\!\alpha &= \int_\id\!\gamma^*\alpha\\ &=\int_0^1\!\sum_{i = 1}^2 \alpha_i(\gamma_i(t))\diff{\gamma_i}{t}\,\dd t\\ &=\int_0^1\! (-\sin 2\pi t)(-2\pi \sin 2\pi t) + \cos 2\pi t (2\pi \cos 2\pi t))\,\dd t\\ &= \cdots\\ &= 2\pi. \end{align*}\end{split}\]

    4. \(\alpha_1 = {\dfrac{1 - x_3}{2}\alpha \in \varOmega^1(S^2\setminus\set{p_S}).}\)

      • これも SymPy で直接計算する：

        alpha1 = ((1 - x3)/2 * alpha).simplify()
print_latex(alpha1)

        結果を記す：

        \[\alpha_1 = \frac{u_1 \dd u_2 - u_2 \dd u_1}{1 + u_1^2 + u_2^2}.\]

      • また、同様にして \({\alpha_2 = \dfrac{1 + x_3}{2}\alpha}\) は \({S^2\setminus\set{p_N}}\) 上の微分形式であることもわかる。そして次問に備えて \({\alpha = \alpha_1 - \alpha_2}\) に注意するとよい。

  • 問題 2.7.5: 前問 の続き

    • \(M_1 = {S^2 \setminus \set{p_S}}\), \(M_2 = {S^2 \setminus \set{p_N}}\)

    • \(M_{12} = {M_1 \cap M_2}\)

    とする（この記号を前問で利用して TeX の手間を軽減したかった）。このとき次の二つのマイヤー・ビエトリス完全系列は同型である：

    \[\begin{split}\begin{CD} \cdots @>>> \H^1(M_1) \oplus \H^1(M_2) @>>> \H^1(M_{12}) @>>> \H^2(S^2) @>>> 0,\\ @.\\ \cdots @>>> \RR \oplus \RR @>>> \RR @>>> \H^2(S^2) @>>> 0. \end{CD}\end{split}\]

    • 前問 の \(\alpha\) は完全形式であり、かつ \({\H^1(M_{12})}\) の生成元である。

      • それぞれ 前問 の小問 2 と 3 による。（生成元の辺りはきちんと説明できるか？）

      • 命題 2.6.2 の周辺の文章をよく読んでおくこと。

    \(\Delta^* {[\alpha|M_{12}]}\) を代表する 2 形式とは何か。

    • \({\dd \alpha}\) を計算していくと、前問 の小問 1 の値が出てくる。このことから次式が成り立つ：

      \[\Delta^* [\alpha|M_{12}] = \frac{1}{2}[\omega|S^2].\]

      なぜなのか、よく考えてみること。

2.8 コンパクト多様体のドラーム・コホモロジー¶

  • 臨界点、ヘッセ行列、非退化、モース関数、モースの補題などのおさらい。

    • 幾何学 I 問題 5.4.8

    • 幾何学 I 補題 5.4.3

    などを参照する。ノートを見返したら難しかったのでまともにやっていなかった。

  • モース関数の gradient flow を用いることで次がわかる：

    \(M\) の開集合 \(N_1, \dotsc, N_k\) で

    • \({\varnothing = N_0 \subset N_1 \subset \dotsb N_k = M}\),

    • \({N_j = N_{j - 1} \cup B_j \quad(0 \lt j \le k)}\)

    なるものがある。ここで各 \(B_j\) は \(n\) 次元開球 \(B^n\) と微分同相であり、

    • \({N_{j - 1} \cap B_j = \varnothing}\)

    • またはある \(m_j\) が存在して \({0 \le m_j \le n - 1}\) のときに \({N_{j - 1} \cap B_j}\) と \({B^{n - m_j} \times S^{m_j}}\) が微分同相である。

  • 定理 2.8.1: コンパクト多様体のドラーム・コホモロジー群は有限次元ベクトル空間である

    証明は \(N_j = {N_{j - 1} \cup B_j}\) についてのマイヤー・ビエトリス完全系列に対する帰納法による。

    \[\begin{split}\begin{CD} @. \cdots @>{(j_1^*,\ j_2^*)}>> \H^{p - 1}(N_{j - 1}) \oplus \H^{p - 1}(B_j) @>{i_1^* - i_2^*}>> \H^{p - 1}(N_{j - 1} \cap B_j)\\ @.\\ @>{\Delta^*}>> \H^p(N_j) @>{(j_1^*,\ j_2^*)}>> \H^p(N_{j - 1}) \oplus \H^p(B_j) @>{i_1^* - i_2^*}>> \H^p(N_{j - 1} \cap B_j)\\ @.\\ @>{\Delta^*}>> \cdots \end{CD}\end{split}\]

    • まず完全系列であることにより \(\H^p(N_j) \cong {\im \Delta^* \oplus \H^p(N_j)}/{\ker (j_1^*,\ j_2^*).}\) ベクトル空間として同型。

      • \({\im \Delta^*} \cong {\H^{p - 1}(N_{j - 1} \cap B_j)}/{\ker \Delta^*}\) が成り立つ（同様）。

        • \({\H^{p - 1}(N_{j - 1} \cap B_j)} \cong {H^{p - 1}(S^{m_j})}.\) 商空間として有限次元ベクトル空間と同型（本節冒頭を参照）。

      • \({\H^p(N_j)}/{\ker (j_1^*,\ j_2^*)} \cong {\im (j_1^*,\ j_2^*)} \subset {H^p(N_{j - 1}) \oplus H^p(B_j).}\) 包含写像の像が有限次元ベクトル空間の部分ベクトル空間であると言っている。最右辺の直和が有限次元であることは、帰納法でわかる。

      以上の二点により \(\H^p(N_j)\) は有限次元である。

  • 注意 2.8.2: モース理論による \(N_j\) への分解はいろいろと有用である。

  • 問題 2.8.3: 幾何学 I 問題 5.4.6 の複素射影空間ファイブレーション 参照

    • 特に参照先の関数 \(f, F\) および臨界点、指数、臨界値についての知識を本問では用いる。

    \(\CC P^n\) のドラーム・コホモロジー群はどのようなものか。

    • まずは誘導された関数 \(F\) により次のような開集合が確定する：

      • \({\varnothing = N_0} \subset N_1 \subset \dotsb N_{n + 1} = \CC P^n\)

      • \(N_j = {N_{j - 1} \cup B_j}\quad{(0 \lt j \le n + 1)}\)

      • \({B_j \cong B^n}\)

      • \({N_{j - 1} \cap B_j} \cong {B^{2n - 2(j - 1) + 1} \times S^{2(j - 1) - 1}\quad(j \ge 2)}\)

    • 次に \(N_j = {N_{j - 1} \cup B_j}\) に関するマイヤー・ビエトリス完全系列を考える。

      \[\begin{split}\begin{CD} @. \cdots @>>> \H^{p - 1}(N_{j - 1}) \oplus \H^{p - 1}(B_j) @>>> \H^{p - 1}(S^{2(j - 1)-1})\\ @.\\ @>>> \H^p(N_j) @>>> \H^p(N_{j - 1}) \oplus \H^p(B_j) @>>> \H^p(S^{2(j - 1)-1})\\ \end{CD}\end{split}\]

      これにより \(\H^p(N_{j - 1})\) が \(\H^p(N_j)\) を定める。なぜならば

      \[\begin{split}\H^p(N_{j - 1}) \cong \begin{cases} \RR & \quad\text{if } p = 0, 2, \dotsc, 2(j - 1)\\ 0 & \quad\text{otherwise} \end{cases}\end{split}\]

      が示されると、

      \[\begin{split}\H^p(N_j) \cong \begin{cases} \RR & \quad\text{if } p = 0, 2, \dotsc, 2j\\ 0 & \quad\text{otherwise} \end{cases}\end{split}\]

      が示される。ゆえに：

      \[\begin{split}\H^p(\CC P^n) \cong \begin{cases} \RR & \quad\text{if } p = 0, 2, \dotsc, 2n\\ 0 & \quad\text{otherwise} \end{cases}\end{split}\]

  • 問題 2.8.4: コンパクト連結多様体 \(M^2\)

    この多様体上にモース関数 \(f\) が存在して、その

    • 極小点が 1 個、

    • 極大点が 1 個、

    • 指数 1 の臨界点が \(k\) 個

    あるとする。このときドラーム・コホモロジー群は次のようになる：

    \[\begin{split}\H^p(M^2) \cong \begin{cases} \RR & \quad\text{if } p = 0\\ \RR^k & \quad\text{if } p = 1\\ \RR & \quad\text{if } p = 2 \end{cases} \quad\text{or } \H^p(M^2) \cong \begin{cases} \RR & \quad\text{if } p = 0\\ \RR^{k - 1}& \quad\text{if } p = 1\\ 0 & \quad\text{if } p = 2 \end{cases}\end{split}\]

    以下証明。例によって開集合列を構成して、完全系列を \(N_j\) と \(M^2\) それぞれについて見ていく。

    • 開集合を構成する：

      • \({\varnothing = N_0} \subset N_1 \subset \dotsb N_{k + 2} = M^2\)

      • \(N_j = {N_{j - 1} \cup B_j}\quad{(0 \lt j \le k + 2)}\)

      • \({B_j \cong B^2}\)

      • \({N_{j - 1} \cap B_j} \cong {B^2 \times S^0\quad{(2 \le j \le k + 1)}}\)

      • \({N_{k + 1} \cap B_{k + 2}} \cong {B^1 \times S^1}\)

    • \(N_j = {N_{j - 1} \cup B_j}\) のマイヤー・ビエトリス完全系列を検討する。完全系列はいつものように描くと三行からなるが、省略。

      • いつものように \(\H^0(\cdot)\) は（局所的）定数関数で代表されるので、一行目の \({i_1^* - i_2^*}\) を行列 \({\begin{pmatrix}1 & -1\\1 & -1\end{pmatrix}}\) で表現できる。

      • \({\H^1(B_j) = 0,}\ {H^1(B^2 S^0) = 0}\) ゆえ \({\H^1(N_j)} \cong {\H^1(N_{j - 1}) \oplus \RR}\) ←二行目

      • \({\H^1(N_1) = 0}\) ゆえ \({\H^1(N_j) \cong \RR^{j - 1}}\)

      • \({\H^2(N_j) = 0\quad{(j \le k + 1)}}\) は容易い。

    • \(M^2 = N_{k + 1} = {N_{k + 1} \cup B_{k + 2}}\) のマイヤー・ビエトリス完全系列を検討する。

      • 一行目の \({i_1^* - i_2^*}\) は全射。

      • 二行目の \({i_1^* - i_2^*}\) は断定できないので場合分け。

        • 全射であれば \({\H^1(N_{k + 2})} \cong {\RR^k/\RR} \cong \RR^{k - 1}\), \({\H^2(N_{k + 2}) = 0}\).

        • ゼロ写像であれば \({\H^1(N_{k + 2})} \cong \RR^k\), \({\H^2(N_{k + 2})} = \RR\).

  • 注意 2.8.5:

    • 問題 2.8.4 の「または～」の側は種数 \(k\) の向き付け不能閉曲面に相当する。

    • \({k = 2g}\) のときの \({\H^p(M)} \cong \RR \text{ or } \RR^{2g} \text{ or } \RR\) に対して、種数 \(g\) の向き付け不能閉曲面。

    • 向き付け可能のとき \(\H^1(M)\) が偶数次元である必要があることは、ポアンカレ双対原理による。
