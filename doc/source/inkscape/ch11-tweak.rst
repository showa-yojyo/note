======================================================================
Chapter 11. Tweak Tool
======================================================================

* 要素選択後、:kbd:`W` or :kbd:`Shift` + :kbd:`F2` で起動。
* 微調整できるモードが色々ある。

Tweaking Paths
----------------------------------------------------------------------

* テキストをパス化したものや、ハッチングに対して適用すると効果的。

Tweaking Colors
----------------------------------------------------------------------

* 偶然に頼って面白い色ができることもある。
