`Modern C++ Tutorial: C++ 11/14/17/20 On the Fly <https://changkun.de/modern-cpp/en-us/>`__
を読んで、思うところをつづっていく。これを読んでいる現時点で私は未だに求職活動か
ら抜け出せないでいる。プログラマーでメシを食っていた時代のもっとも得意な言語は
C++ であるのだが、C++11 コンパイラーが登場する頃には C++ の業務がなくなってい
た。その結果、私の C++ の能力が C++98 で止まっている。これは面接で弱みになる。そ
こで、早急に現代の C++ 事情を身につけるのが望ましいと判断。良さそうな教材を探し
たところ、本書を発見した。少しでも現代に追いつきたい。そして C++ をいじれる現場
に潜り込みたい。
